    Section 2.4 Advanced Decompositions

    Subsection 2.4.1 Linear Factors

    Example 2.4.1.1.

    Heaviside’s method, as we have used it so far, will not work on the rational function \(\frac{x^2}{(x-2)(x-3)}\text{.}\) If you try to apply it you’ll get
    \begin{equation*} \frac{x^2}{(x-2)(x-3)} = \frac{-4}{x-2}+\frac{9}{x-3} \end{equation*}
    which is wrong. Try it and see.
    The difficulty here is that, as a practical matter, the PFD requires that we want the degree of the numerator of our rational function to be strictly less than the degree of the denominator. (This is somewhat analogous to putting a fraction in “lowest terms.”) In this example the degree of the numerator is
    \begin{equation*} \deg\left((x-2)(x-3)\right)=\deg\left(x^2-5x+6\right)=2 \end{equation*}
    which is the same as the degree of the numerator: \(\deg\left(x^2\right)=2\text{.}\) So the PFD, as we’ve used it so far, doesn’t apply.
    At first this looks like it will be a major obstacle but it really isn’t too bad. There are two ways we can proceed from here.
    Dividing First: The first is simply to do the polynomial division indicated. In practice that means multiplying out the denominator first and then dividing:
    \begin{align} \frac{x^2}{(x-2)(x-3)} = \frac{x^2}{x^2-5x+6} \amp{}= 1+ \frac{5x-6}{x^2-5x+6}\notag\\ \amp{}=1+\frac{5x-6}{(x-2)(x-3)}\tag{2.41} \end{align}
    From there we can compute the PFD of \(\frac{5x-6}{(x-2)(x-3)}\) as before, finally obtaining
    \begin{equation*} \frac{x^2}{x^2-5x+6} = 1+\frac{-4}{x-2}+\frac{9}{x-3}\text{.} \end{equation*}
    Dividing Last: The second is to factor out the numerator:
    \begin{equation*} \frac{x^2}{(x-2)(x-3)} = x^2 \left[ \frac{1}{(x-2)(x-3)} \right] \text{,} \end{equation*}
    compute the PFD of \(\frac{1}{(x-2)(x-3)}\) as before. This gives
    \begin{align} \frac{x^2}{(x-2)(x-3)} \amp{}= x^2 \left[ \frac{-1}{x-2}+\frac{1}{x-3} \right]\notag\\ \amp{}=\frac{-x^2}{x-2}+\frac{x^2}{x-3}\tag{2.42}\\ \amp{}= -x-2-\frac{4}{x-2} +x+3+\frac{9}{x-3}\notag\\ \amp{}=1-\frac{4}{x-2}+\frac{9}{x-3}.\notag \end{align}
    Whether you divide first or last is entirely up to you. We (the authors) prefer dividing last because it doesn’t involve factoring and then un–factoring the denominator, and because we only ever have to divide by a linear expression, which we find less burdensome.
    However, we do advise that you pick one method that you like and use it consistently, at least until you have fully mastered the Partial Fraction Decomposition.

    Drill 2.4.1.2.

    Confirm that the decompositions in equations (2.41) and (2.42) are correct.

    Problem 2.4.1.3.

    Despite our personal preference for dividing last there are legitimate reasons to divide first. For example show that the expression \(\frac{x}{x-1}\) cannot be decomposed by the “dividing last” method described in Example 2.4.1.1, but it can be decomposed by the “dividing first” method. Are we really “decomposing” the fraction in this case. Explain.

    Note to self.

    We need a representative and instructive set of problems here.

    Example 2.4.1.4.

    Heaviside’s Cover–up Method will also fail on this function:
    \begin{equation*} \frac{x^2+x-3}{x^2(x-1)}\text{.} \end{equation*}
    If you apply it you’ll get
    \begin{equation*} \frac{x^2+x-3}{x^2(x-1)} = \frac{3}{x^2} + \frac{-1}{x-1} \end{equation*}
    which is incorrect. Check it and see.
    The problem here is the factor \(x^2\) in the denominator. If you look back at the problems and examples we’ve looked at so far you’ll see that each distinct factor which appeared in the denominator of the fraction on the left side also appeared in one of the denominators on the right side. In this case the distinct factors are \(x^2\) and \(x-1\) so the decomposition will be
    \begin{equation*} \frac{x^2+x-3}{x^2(x-1)} = \frac{\text{something}}{x^2} + \frac{C}{x-1} \text{.} \end{equation*}
    But now the question is, “What is the something?” Up until now none of the distinct factors in the denominator appeared with a power higher than one. So a factor of \(x^2\) is altogether new and different.
    Looking back at our earlier, and simpler, examples we see that every case the terms on the left were rational functions where the degree of the numerator (which was always \(0\) in those examples) was one less than the degree of the denominator (which was always \(1\) in those examples). If we keep that pattern here then it appears that the decomposition must be
    \begin{equation*} \frac{x^2+x-3}{x^2(x-1)} = \frac{Ax+B}{x^2} + \frac{C}{x-1} \end{equation*}
    because
    \begin{equation*} \deg (Ax+B)=1 \end{equation*}
    and
    \begin{equation*} \deg(x^2) =2\text{.} \end{equation*}
    We can break this up a bit more by observing that
    \begin{equation*} \frac{Ax+B}{x^2} =\frac{A}{x}+\frac{B}{x^2} \end{equation*}
    so we need to find \(A\text{,}\) \(B\text{,}\) and \(C\) where
    \begin{equation*} \frac{x^2+x-3}{x^2(x-1)} = \frac{A}{x}+\frac{B}{x^2} + \frac{C}{x-1} \text{.} \end{equation*}
    The values \(B=3\text{,}\) and \(C=-1\) can still be found using the HCUM, so we have
    \begin{equation} \frac{x^2+x-3}{x^2(x-1)} = \frac{A}{x} + \frac{3}{x^2}- \frac{1}{x-1}\text{.}\tag{2.43} \end{equation}
    To find \(A\) recall that equation (2.43) is an identity so it is true regardless of the value of \(x\text{.}\) Since \(A\) is the only parameter left we can find it by setting \(x=2\) (or anything else). This gives \(\frac{2}{2^2(2-1)} = \frac{A}{2} + \frac{3}{2^2}- \frac{1}{2-1} \text{,}\) or \(A=2\text{.}\) So the full decomposition is
    \begin{equation*} \frac{x^2+x-3}{x^2(x-1)} = \frac{2}{x}+ \frac{3}{x^2}-\frac{1}{x-1}\text{.} \end{equation*}

    Find \(B\text{,}\) \(C\text{,}\) and \(F\text{.}\)
    \begin{align*} \frac{1}{x^2(x-1)(x-3)^3}=\amp{} \frac{Ax+B}{x^2} + \frac{C}{x-1}\\ \amp{}+ \frac{D(x-2)^2+E(x-3)+F}{(x-3)^3} \end{align*}
    (b)

    \begin{equation*} \displaystyle \frac{x}{(x-1)^2(x+2)^2}= \frac{A(x-1)+B}{(x-1)^2} +\frac{C(x+2)+D}{(x+2)^2} \end{equation*}

    In Problem 2.4.1.6 we only asked you to find the coefficients that are attainable using the HCUM. The others will have to be computed by the method we’ll describe in the next example.

    Example 2.4.1.7. The PFD With Repeated Linear Factors.

    Let’s finish up the decomposition of \(\frac{x}{(x-1)^2(x+2)^2}\) from part (b) of Problem 2.4.1.6. From the HCUM You should have found that \(B=\frac19\) and \(D=\frac{-2}{9}\) so that
    \begin{equation*} \displaystyle \frac{x}{(x-1)^2(x+2)^2}= \frac{A}{x-1} + \frac{\frac19}{(x-1)^2} +\frac{C}{x+2} -\frac{\frac29}{(x+2)^2} \end{equation*}
    When we reached this point in Example 2.4.1.4 our next step was to simply choose a value for \(x\text{.}\) We could do that here as well but we’d have to choose two values for \(x\) thereby generating a pair of equations to solve and we’ve already seen that there are more efficient methods for computing the PFD than solving a system of equations.
    Instead we’ll isolate all of the terms with our unknowns on one side of the equation, thus:
    \begin{equation} \displaystyle \frac{x}{(x-1)^2(x+2)^2} - \frac{\frac19}{(x-1)^2} +\frac{\frac29}{(x+2)^2} = \frac{A}{x-1} +\frac{C}{x+2}\tag{2.44} \end{equation}
    It is clear that the right side of equation (2.44) is the PFD of some rational function whose denominator is \((x-1)(x+2)\text{.}\) That is, if we were asked to begin the process of computing the PFD of
    \begin{equation*} \frac{P(x)}{(x-1)(x+2)} \end{equation*}
    for some polynomial \(P(x)\) then our first step would be to write down the right side of equation (2.44). Therefore it must be that
    \begin{equation} \displaystyle \frac{x}{(x-1)^2(x+2)^2} - \frac{\frac19}{(x-1)^2} +\frac{\frac29}{(x+2)^2} = \frac{P(x)}{(x-1)(x+2)}\tag{2.45} \end{equation}
    for some polynomial \(P(x)\text{,}\) and if we can find that polynomial we can compute \(A\) and \(C\) by the HCUM.
    We can find \(P(x)\) by essentially the same method we used to find the constant \(A\) in Example 2.4.1.4. First we clear the fraction on the right
    \begin{equation} \frac{x}{(x-1)(x+2)} - \frac19 \left( \frac{x+2}{x-1} \right) +\frac29 \left( \frac{x-1}{x+2} \right) = P(x)\tag{2.46} \end{equation}
    Since the left side is guaranteed to reduce to the polynomial it is tempting to find a common denominator and add the three fractions on the left together. Since we have to end up with the polynomial \(P(x)\) it must be that once the addition is done there will be a factor in the top that will cancel with the common denominator.
    That will work. Give it a try if you want. Good luck.
    But we’re trying to find the most efficient method we can so instead we’re going to make judicious use of the knowledge that we must end up with a polynomial.
    First think about the expression in the second term on the left: \(\frac{x+2}{x-1}\text{.}\) If we perform this division we get
    \begin{equation*} \frac{x+2}{x-1} = 1 +\frac{3}{x-1}\text{.} \end{equation*}
    Similarly if we perform the division in the third term we get \(\frac{x-1}{x+2}= 1-\frac{3}{x+2}\text{.}\) Putting these back into equation (2.46) we have
    \begin{equation*} \frac{x}{(x-1)(x+2)} - \frac19 \left( 1 +\frac{3}{x-1} \right) +\frac29 \left( 1-\frac{3}{x+2} \right) = P(x) \end{equation*}
    or
    \begin{equation*} \frac{x}{(x-1)(x+2)} - \frac19 -\frac{\frac13}{x-1} +\frac29 -\frac{\frac23}{x+2} = P(x) \text{.} \end{equation*}
    Since \(P(x)\) must be a polynomial it follows that \(\frac{x}{(x-1)(x-2)}-\frac{\frac13 }{x-1}-\frac{\frac23 }{x+2}\) (everything that isn’t a polynomial) must sum to zero. So we can ignore them and conclude that our polynomial is given by the other terms. So
    \begin{equation*} P(x)=-\frac19 + \frac29 =\frac19\text{.} \end{equation*}
    To finish the decomposition we observe that
    \begin{equation*} \frac{\frac19 }{(x-1)(x+2)} = \frac{A}{x-1}+\frac{C}{x+2} \end{equation*}
    and from the HCUM we see that \(A=\frac{1}{27} \) and \(C=-\frac19\text{.}\)
    Having found all of the constants we conclude that
    \begin{equation*} \frac{x}{(x-1)^2(x+2)^2}= \frac{\frac{1}{27}}{x-1} + \frac{\frac19 }{(x-1)^2} -\frac{\frac19}{x+2} -\frac{\frac29 }{(x+2)^2} \text{.} \end{equation*}
