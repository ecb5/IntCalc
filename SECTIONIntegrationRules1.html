    Section 3.2 Rules for Integration

    Since integration can be thought of as the reverse of differentiation it is not too surprising to learn that each differentiation rule has its counterpart as an integration rule. The following list illustrates some of the rules we already know from Differential Calculus in both their differential and integral form.
    \begin{align} \text{Differentiation}\text{ Rule}\amp{}\amp{}\text{Integration} \text{ Rule}\notag \end{align}
    \begin{align} \dx(\text{constant})=0;\amp{}\amp{}\int0\dx{x}=C\tag{3.5} \end{align}
    \begin{align} \dx(f+g)=\dx{f}+\dx{g};\amp{}\amp{}\int\dx(f+g)=f+g+C\tag{3.6} \end{align}
    \begin{align} \text{If }a\text{ is constant, then } \notag\\ \dx(ay)=a\dx(y);\amp{}\amp{}\int a\dx{y}=a\int\dx{y}=ay+C\tag{3.7} \end{align}
    \begin{align} \dx(x^{n+1})=(n+1)x^n\dx{x} ;\amp{}\amp{}\int x^n\dx{x} = \frac{x^{n+1}}{n+1}+C, n\neq -1\tag{3.8} \end{align}
    \begin{align} \dx(\ln(x))=x^{-1}\dx{x} ;\amp{}\amp{}\int x^{-1}\dx{x} = \ln\abs{x}+C\tag{3.9} \end{align}
    \begin{align} \dx(e^x)= e^x\dx{x} ;\amp{}\amp{}\int e^x\dx{x} = e^x+C\tag{3.10} \end{align}
    \begin{align} \dx(\sin(x))=\cos(x)\dx{x} ;\amp{}\amp{}\int \cos(x)\dx{x} = \sin(x) +C\tag{3.11} \end{align}
    \begin{align} \dx(\cos(x))=-\sin(x)\dx{x} ;\amp{}\amp{}\int \sin(x)\dx{x} = -\cos(x) +C\tag{3.12} \end{align}
    \begin{align} \dx(\tan(x))=\sec^2(x)\dx{x} ;\amp{}\amp{}\int\sec^2(x)\dx{x} = \tan(x)+C\tag{3.13} \end{align}
    \begin{align} \dx(\sec(x))= \sec(x)\tan(x)\dx{x} ;\amp{}\amp{}\int\sec(x)\tan(x)\dx{x}=\sec(x)+C\tag{3.14} \end{align}
    \begin{align} \dx(\inverse\sin(x))= \frac{1}{\sqrt{1-x^2}}\dx{x} ;\amp{}\amp{}\int\frac{1}{\sqrt{1-x^2}}\dx{x}=\inverse\sin(x)+C\tag{3.15} \end{align}
    \begin{align} \dx(\inverse\tan(x))= \frac{1}{1+x^2}\dx{x} ;\amp{}\amp{}\int\frac{1}{1+x^2}\dx{x}=\inverse\tan(x)+C\tag{3.16} \end{align}
    \begin{align} \dx(\inverse\sec(x))= \frac{1}{x\sqrt{x^2-1}}\dx{x} ;\amp{}\amp{}\int\frac{1}{x\sqrt{x^2-1}}\dx{x}=\inverse\sec(x)+C\tag{3.17} \end{align}

    Drill 3.2.0.1.

    Integration Rules (3.6) and (3.7) appear to violate Integration Rule (3.5), don’t they? Isn’t it true that
    \begin{equation*} \int\dx(f+g)= \int \dx{f} +\int \dx{g} = f+C_1+g+C_2 \end{equation*}
    where \(C_1\) and \(C_2\) are arbitrary constants, and that
    \begin{equation*} a\int\dx{y}=a(y+c)=ay+aC \text{.} \end{equation*}
    Explain how Integration Rules (3.6) and (3.7) are legitimate as stated.
    In light of Drill 3.2.0.1, it is common practice integrate each term of a differential and then combine all of the arbitrary constants into a single constant.

    Example 3.2.0.2.

    For example, consider
    \begin{align*} \int\left(5x^2-3\sqrt{x}+\frac{2}{1+x^2}\right)\amp{}\dx{x}\\ \amp{}=5\int x^2\dx{x}-3\int x^\frac12 \dx{x}+2\int\frac{1}{1+x^2}\dx{x}\\ \amp =5\left(\frac{x^3}{3}\right) +C_1- \frac{3\left(x^{\frac12}\right)}{\frac32} + C_2 + 2\inverse\tan(x)+C_3\\ \amp =\frac53x^3-2x^{\frac32}+2\inverse\tan(x)+C\text{,} \end{align*}
    where \(C=C_1+C_2+C_3\text{.}\)
    As with differentiation, when you get more comfortable with integration, you will probably do many of these steps in your head. But that ability will only come with practice.

    Drill 3.2.0.3.

    Use Integration Rules (3.5)–(3.17) to find the most general antiderivative.
    1. \(\displaystyle \int t^2\dx{t}\)
    2. \(\displaystyle \int 3t-2\dx{t}\)
    3. \(\displaystyle \int 7\dx{t}\)
    4. \(\displaystyle \int \sin(t)\dx{t}\)
    5. \(\displaystyle \int 5\cos(t)\dx{t}\)
    6. \(\displaystyle \int \sec(t)\tan(t)\dx{t}\)
    7. \(\displaystyle \int t + e^t\dx{t}\)
    8. \(\displaystyle \int \frac{1}{1+\theta^2}\dx{\theta{}}\)
    Conspicuously absent from our list of rules are the integral form of the Product Rule, the Quotient Rule, and the Chain Rule. The reverse of the Quotient Rule is rather limited in its applicability and typically is not used. Reversing the others can be quite useful but we are not quite ready to discuss them in depth.

    DIGRESSION: A Few Comments on Integration Rule (3.9).

    First, notice that Integration Rule (3.8) will work if the exponent has any value except \(n=-1\text{.}\) If we try to apply it we’ll get
    \begin{equation*} \int \inverse{x}\dx{x} = \frac{x^{1-1}}{1-1} = \frac{1}{0} \end{equation*}
    which we know is meaningless.
    For this single exception we have to turn to Integration Rule (3.9): \(\int\frac1x\dx{x}=\ln\abs{x}+C{}\text{.}\)
    Does it seem strange to you that there would be this single exception to Integration Rule (3.8)? Us too.
    Also notice the absolute value in the formula:
    \begin{equation*} \int\inverse x\dx{x}=\ln\abs{x}+C. \end{equation*}
    We saw in [cross-reference to target(s) "DIGRESSIONAbsValSqrt" missing or not unique] that an alternative definition of the absolute value function is
    \begin{equation*} \abs{x}=\sqrt{x^2}\text{.} \end{equation*}

    Problem 3.2.0.4.

    Show that if we differentiate the definition \(\ln(x) =\sqrt{x^2}\) we still obtain the formula
    \begin{equation*} \dx(\ln\abs{x})=\frac1x \dx{x}\text{.} \end{equation*}
    Hint.
    Be careful that you do not try to take the logarithm of a negative number.
    It is common (though incorrect) practice to disregard the absolute value and write \(\int\inverse x\dx{x}=\ln(x)+C\text{,}\) even though this is only valid for \(x\gt{0}\text{.}\) We will also adopt this practice but you should always bear in mind that there are situations where the absolute value must be used.
