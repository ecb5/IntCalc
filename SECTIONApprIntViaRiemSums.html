    Section 22.3 Approximating Integrals with Riemann Sums

    As we saw in the last section, representing a function as a power series was a powerful tool for computing both indefinite and definite integrals and for solving differential equations. As the focus of integral calculus turned more toward computing areas, volumes, arc lengths, etc. via definite integrals, a more natural approach to approximating them came into view. This idea was to replace an infinite sum of infinitesimals with a finite sum of finite quantities. More specifically, instead of dividing the interval into infinitely small pieces \(\dx{x}\) or \(\dx{y}\) and integrating differentials together, we could divide the interval into a finite number of pieces and create a finite sum. This could be used to approximate the integral itself and, in theory, if the lengths \(\Delta x\) are made smaller (meaning more terms to sum together) then the approximation would be more accurate. In fact, in a rigorous formulation of integrals, we could take the limit of these finite sums and it should converge to what we call the definite integral.
    🔗
    For example, suppose we wanted to approximate
    🔗
    \begin{equation*} \int^1_{x=0}{e^{-x^2}}\dx{x} \end{equation*}
    🔗
    This would be the area under the curve \(y=e^{-x^2}\) from \(x=0\) to \(x=1\text{.}\) We could divide the interval \([0,1]\) into a finite number of equal pieces, say 10. The length of each subinterval would be \(\Delta x=\frac{1}{10}\text{.}\) We could then choose a point \(x^*\) in each subinterval and create the finite sum
    🔗
    \begin{equation*} e^{-{\left(x^*_1\right)}^2}\Delta x+e^{-{\left(x^*_2\right)}^2}\Delta x+\dots +e^{{-\left(x^*_{10}\right)}^2}\Delta x=\sum^{10}_{n=1}{e^{{-\left(x^*_n\right)}^2}\Delta x} \end{equation*}
    🔗
    It is hard to digest what is going on without a picture so here is the situation.
    🔗
    As you can see, the sum
    \begin{align*} e^{{-\left(x^*_1\right)}^2}\Delta x+e^{{-\left(x^*_2\right)}^2}\Delta x+\dots +e^{{-\left(x^*_{10}\right)}^2}\amp{}\Delta x\\ =\amp{}e^{{-\left(x^*_1\right)}^2}\mathrm{.1}+e^{{-\left(x^*_2\right)}^2}\mathrm{.1}+\dots +e^{{-\left(x^*_{10}\right)}^2}\mathrm{.1}\\ =\amp{}0.1\sum^{10}_{n=1}{e^{{-\left(x^*_n\right)}^2}} \end{align*}
    represents the (finite) sum of the areas of boxes and we can use this to approximate the area under the curve given by \(\int^1_{x=0}{e^{-x^2}}\dx{x}\text{.}\) We are free to choose \(x^*_n\) to be any value inside each subinterval. For example, we can choose the midpoints in each interval: \(0.05,\) \(0.15,\) \(0.25,\) \(0.35,\) \(0.45,\) \(0.55,\) \(0.65,\) \(0.75,\) \(0.85,\) \(0.95.\) This finite sum can be calculated on a computer. Furthermore, if we have enough computing power, we could divide the interval \([0,1]\) into as many subintervals as we wish, and they don’t even need to be the same size. A sum such as above is called a Riemann sum named after Bernhard Riemann (1826-1866) who took limits of these sums as \(\Delta x\to 0\) to carefully define what a definite integral is without having to resort to differentials. We’ll get more into this story in the theory part, but for now let’s focus on approximating integrals.

    DIGRESSION: Simpson’s Rule and Error Bounds.

    If you look at the midpoint rule geometrically, you will notice that we are replacing the curve \(y=f(x)\) on each subinterval \(\left[x_{k-1},x_k\right]\) with the horizontal line \(y=f(x^*_k)\) where \(x^*_k\) is the midpoint of the subinterval. Said in a different way, on each subinterval, we are replacing the curve with the zero-degree polynomial \(y=f(x^*_k)\text{.}\) In the trapezoidal rule, we are replacing the curve \(y=f(x)\) on each subinterval \([x_{k-1},x_k]\) with the first-degree polynomial (line) passing through the points \(\left(x_{k-1},f\left(x_{k-1}\right)\right)\) and \(\left(x_k,f\left(x_k\right)\right)\text{.}\) What if we were to replace the curve \(y=f(x)\) on each subinterval with an appropriate second-degree polynomial? Since a zero-degree polynomial requires the single point \(\left(x^*_k,f\left(x^*_k\right)\right)\) to determine and the trapezoidal rule requires two points \(\left(x_{k-1},f\left(x_{k-1}\right)\right)\) and \(\left(x_k,f\left(x_k\right)\right)\) to determine, it stands to reason that replacing the curve \(y=f(x)\) with a parabola on each subinterval would require three points. The natural choice would be the endpoints and midpoint of each subinterval. With that in mind, let’s just consider a function \(y=f\left(x\right)\) defined on \([a,b]\) and replace it by the parabola \(y=P(x)\) passing through the points \(\left(a,f\left(a\right)\right), \left(b,f\left(b\right)\right)\) and \(\left(m,f\left(m\right)\right)\) where \(m=\frac{a+b}{2}\) is the midpoint of the interval \([a,b]\text{.}\) Here is a picture depicting this.
    🔗
    To make things a bit easier, let’s shift the graph over so that \(m=0. \)In this case we will denote \(h=b-m=\frac{b-a}{2}\) so that \(a-m=\frac{a-b}{2}=-h.\)

    🔗
    The approximation
    \begin{equation*} \int^b_{x=a}{f\left(x\right)\dx{x}}\approx \int^h_{x=-h}{P\left(x\right)} dx=\frac{h}{3}\left[f\left(a\right)+4f\left(m\right)+f\left(b\right)\right] \end{equation*}
    where \(h=\frac{b-a}{2}\) and \(m=\frac{a+b}{2}\) is called Simpson’s 1/3 rule named after Thomas Simpson (1710-1761) who included a version of it in his book Mathematical Dissertations (1743). Versions of it were known to Cavalieri, Gregory, and Kepler in the 17th century, but Simpson’s book popularized it.
    🔗
    We can use this special case where we divided the interval \([a,b]\) into 2 equal subintervals to the case where we divide an even number \(n=2m\) of subintervals each of length \(\Delta x=\frac{b-a}{2m}\text{.}\) We need an even number of subintervals so we can apply Simpson’s 1/3 rule to each pair of adjacent intervals. It’s easy to get lost in the notation, so here is a picture of the case where \(m=3\) and we are dividing the interval \([a,b]\) into \(6\) subintervals of width \(\Delta x=\frac{b-a}{6}\text{.}\)

    An obvious question regarding any approximation is how accurate it is. This question is typically addressed in a numerical analysis course, but we can state a few things here. First of all, we can see geometrically that the midpoint rule and trapezoidal rule will provide an exact answer for a linear function \(f\left(x\right)=Mx+B\text{.}\)
    🔗
    For more general curves, we have the following results (stated without proof).
    🔗
    Error Bounds Suppose \(\left|f^{\prime\prime}\left(x\right)\right|\le K\) on \([a,b]\text{,}\) if \(E_M\) and \(E_T\) represent the errors in the midpoint and trapezoidal rules with \(n\) equal subintervals, then
    🔗
    \begin{align*} \left|E_M\right|\le \frac{K{\left(b-a\right)}^3}{24n^2}\amp{}\amp{} \left|E_T\right|\le \frac{K{\left(b-a\right)}^3}{12n^2} \end{align*}
    🔗

    Theorem 22.3.0.12. An Error Bound for Simpson’s rule.

    Suppose that \(\left|f^{\left(4\right)}\left(x\right)\right|\le K\) on \([a,b]\text{.}\) Then the error \(E_S\) when approximating using Simpson’s rule with \(n=2m\) equal subintervals satisfies
    \begin{equation*} \left|E_S\right|\le \frac{K{\left(b-a\right)}^5}{180n^4} \end{equation*}

    🔗
