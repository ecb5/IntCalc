    Section 6.1 Initial Value Problems (IVPs)

    Let’s step back for a moment. Recall the definition of an Initial Value Problem from Definition 1.10. An IVP consists of two parts:
    1. The differential equation (for example, \(\dfdx{y}{x}=2x\)),
    2. The initial value. (So called because when the independent variable is time we usually know the value of our function initially.)
    A common way to create new functions using Calculus is by defining the new function to be the solution(s) of a specified IVP.

    Example 6.1.

    We’ve seen that when we differentiate the formula \(y=x^2\) we get the differential equation \(\dx{y} = 2x\dx{x},\) or
    \begin{equation} \dfdx{y}{x}=2x.\tag{6.1} \end{equation}
    If we did not already know that the function \(y(x)=x^2\) satisfies equation (6.1) we could give a name to the solution (sqr\((x)\text{,}\) perhaps?) and by fiat, define sqr\((x)\) to be whatever function solves this equation. But there is a problem. The solution of equation (6.1) is a multifunction, remember? To choose a single branch we need to impose an initial condition.

    Problem 6.2.

    Solve each of the following IVPs. Graph your solutions:
    1. \(\displaystyle \dfdx{y}{x}=2x,\ \ \ y(0)=0\)
    2. \(\displaystyle \dfdx{y}{x}=2x,\ \ \ y(0)=1\)
    3. \(\displaystyle \dfdx{y}{x}=2x,\ \ \ y(0)=-10\)
    4. \(\displaystyle \dfdx{y}{x}=2x,\ \ \ y(1)=2\)
    5. \(\displaystyle \dfdx{y}{x}=2x,\ \ \ y(-1)=1\)
    6. \(\displaystyle \dfdx{y}{x}=x,\ \ \ y(0)=1\)
    7. \(\dfdx{y}{x}=2x,\ \ \ y(x_0)=y_0\) where \(x_0\) and \(y_0\) are fixed, but unspecified constants.
    Simply naming the function tells us nothing about it. However the IVP itself can give us a some insight into the nature of the solution. For example since \(2x\lt 0\) when \(x\lt 0\text{,}\) the slope of the graph of \(y(x)\) is clearly also negative when \(x\lt 0\text{,}\) Similarly, the slope of the graph of \(y(x)\) is positive when \(x\gt 0\text{.}\)
    Of course we already knew that since we have a formula for \(y(x)=\text{sqr}(x)=x^2\text{.}\) This won’t be true in the next section.
