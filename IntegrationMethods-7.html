    Section 2.6 Back to Logistic Growth: Partial Fractions

    Subsection 2.6.1 Intro

    Recall that in Section 6.4 of “Differential Calculus” we had this example for logistic growth
    \begin{equation} \dfdx{y}{t}=.01y\left(1-\frac{y}{100}\right)=0.0001(100-y)\text{.}\tag{2.11} \end{equation}
    In that chapter, we analyzed this growth rate to draw a qualitative graph of \(y=y(t)\text{,}\) but we could not produce specific values of \(t\) for where things happened, because we didn’t have a formula for \(y(t)\text{.}\) We now have the techniques needed to find \(y(t)\text{.}\)

    Note to Self.

    Turn this into a PreTeXt reference.
    First we separate the variables:
    \begin{equation*} \frac{\dx{y}}{y(100-y)}=0.0001y\dx{t} \text{.} \end{equation*}
    To integrate the left-hand side, we could expand the denominator to complete the square and apply the appropriate trigonometric substitution.
    As we’ve mentioned before, all mathematical models are simplifications of reality, and typically are modified to take into account more complexities. For example, consider the following modification of the logistic model above
    \begin{equation*} \frac{dy}{dt}=.01y\left(1-\frac{y}{100}\right)\left(\frac{y}{10}-1\right)=0.00001y(100-y)(y-10) \end{equation*}
    In this case, \(10\) is called the minimum viability level of the population. (Why?) Solving this would require that we compute the integral
    \begin{equation*} \int \frac{\dx{y}}{y(100-y)(y-10)}\text{.} \end{equation*}
    This form of this integral does not suggest any obvious trigonometric substitution so we will need a new idea.
    The next technique is wholly algebraic in nature. The trick to try and separate the integrand into partial fractions each of which will be easier to integrate than the original. For example, consider the expression \(\frac{1}{y(100-y)}\text{.}\) With begin by making a reasonable, educated guess that such a fraction can be written as \(\frac{1}{y(100-y)}=\frac{A}{y}+\frac{B}{100-y}\text{,}\) where \(A\) and \(B\) are constant to be determined. We can see if this guess pans out by combining the partial fractions to obtain the original.
    \begin{equation*} \frac{A}{y}+\frac{B}{100-y}=\frac{A(100-y)+By}{y(100-y)}=\frac{100A+(B-A)y}{y(100-y)} \end{equation*}
    Comparing this with \(\frac{1}{y(100-y)}\) we see that if they are to be equal we must have \(100A=1\) and \(B-A=0\text{.}\) Therefore \(A=B=\frac{1}{100}\text{.}\) Thus
    \begin{equation*} \frac{1}{y(100-y)}=\frac{1/100}{y}+\frac{1/100}{100-y}\text{.} \end{equation*}
    Substituting into the original integral we see that
    \begin{align*} \int\frac{\dx{y}}{y(100-y)}\amp amp;=\int\frac{1/100}{y}\dx{y}+\int\frac{1/100}{100-y}\dx{y}\\ \amp amp;=\frac{1}{100}\left(\int\frac{\dx{y}}{y}+\int\frac{\dx{y}}{100-y}\right) \end{align*}
    and so
    \begin{gather} \int\frac{\dx{y}}{y(100-y)}=\frac{1}{100}\left(\ln(y)-\ln(100-y)\right)+C\tag{2.12} \end{gather}

    Comment.

    This isn’t really a guess. There is a theorem from algebra that says that there will always be constants \(A\) and \(B\) such that \(\frac{1}{y(100-y)}=\frac{A}{y}+\frac{B}{100-y}\) but since we haven’t discussed this yet we’ll just roll with it for now.

    Problem 2.45.

    (a)
    Use equation (2.12) to find \(y\) for the logistic growth model with the initial condition \(y(0)=0\)
    \begin{equation*} \dfdx{y}{t}=0.01y\left(1-\frac{y}{100}\right)\hskip5mm y(0)=0 \end{equation*}
    for the case when \(y\lt 100\text{.}\)
    (b)
    Solve the same problem for the case where \(y\gt 100\text{.}\)
    Comment.
    This is a spot where you need to look at the absolute value in the logarithm. We told you it comes up occasionally.

    Subsection 2.6.2 The Partial Fraction Decomposition

    Now that we’ve seen that decomposing a ratio of polynomials (these are called rational functions) into a sum of partial fractions is possible, we’ll look for a systematic way of doing this for any rational function. Our method will hinge on the following theoretical, algebraic fact.
    Suppose we have two polynomials \(p(x)\) and \(q(x)\) which have no common factors (other than constants). Then there are polynomials \(r(x)\) and \(s(x)\) such that
    \begin{equation*} r(x)\cdot{}p(x)+s(x)\cdot{}q(x)=1\text{.} \end{equation*}

    Example 2.46.

    For example notice that if \(a\) and \(b\) are constants then \(x+a\) and \(x+b\) have no common factors. Since \((x+a)-(x+b)=a-b\) it follows that
    \begin{equation*} \frac{1}{a-b}(x+a)+\frac{-1}{a-b}(x+b)=1\text{.} \end{equation*}
    Comment.
    Notice that since \(\frac{1}{a-b}\) and \(\frac{-1}{a-b}\) are constants they are polynomials of degree zero.
    Our approach to the problem in example 2.46was somewhat ad-hoc. We will eventualy need a more systematic approach but it will be useful to look at some more examples first. So for now we will use an (educated) guess and check method. Notice that in the example 2.46 we had
    \begin{align*} \frac{1}{(x+a)(x+b)}\amp amp;=\frac{\frac{1}{a-b}(x+a)+\frac{-1}{a-b}(x+b)}{(x+a)(x+b)}\\ \amp amp;= \frac{\frac{1}{a-b}}{x+b}+\frac{\frac{-1}{a-b}}{x+a}\\ \amp amp;=\frac{\frac{1}{a-b}}{x+b}-\frac{\frac{1}{a-b}}{x+a} \end{align*}
    It is generally true that if \(r(x)\cdot p(x+s(x)\cdot q(x)=1\) then
    \begin{equation*} \frac{1}{p(x)\cdot{}q(x)}=\frac{r(x)\cdot{}p(x)+s(x)\cdot{}q(x)}{p(x)\cdot{}q(x)}= \frac{s(x)}{p(x)}+\frac{r(x)}{q(x)}\text{.} \end{equation*}
    Multiplying by \(t(x)\) we have
    \begin{equation*} \frac{t(x)}{p(x)\cdot{}q(x)}= \frac{t(x)\cdot s(x)}{p(x)}+\frac{t(x)\cdot{}r(x)}{q(x)}\text{.} \end{equation*}
    Setting \(A(x)=t(x)\cdot{}s(x)\) and \(B(x)=t(x)\cdot r(x)\) gives
    \begin{gather*} \frac{t(x)}{p(x)\cdot{}q(x)}= \frac{A(x)}{p(x)}+\frac{B(x)}{q(x)}\text{.} \end{gather*}
    Everything we’ve just shown assumes that if \(p(x)\) and \(q(x)\) are known then \(r(x)\) and \(s(x)\) can be found so that \(r(x)\cdot p(x)+ s(x)\cdot q(x)=1\text{.}\) For arbitrary functions this is a very restrictive condition. However as we observed above, if \(p(x)\) and \(q(x)\) are polynomials having only trivial common factors then the existence of \(r(x)\) and \(s(x)\) is guaranteed. Thus for polynomials the Partial Fraction Decomposition is also guaranteed.

    Mathematical Language.

    The word “trivial” is very context dependent in mathematics. In this context a “trivial common factor would be a constant.” For example, \(2\) is a trivial common factor of \((2x-4)\) and \(2x+6\text{.}\)
    We formalize this in the following theorem.

    Theorem 2.47. The Partial Fraction Decomposition (PFD).

    If \(P(x)\text{,}\) \(Q(x)\text{,}\) and \(R(x)\) are polynomials having no non-trivial common factors with \(\deg(P)\lt\deg(Q)+ \deg(R)\) then there exist polynomials \(A(x)\) and \(B(x)\text{,}\) \(\deg{A}\lt\deg{Q}\) and \(\deg(B)\lt\deg(R)\text{,}\) such that
    \begin{equation*} \frac{P(x)}{Q(x)\cdot{}R(x)} = \frac{A(x)}{Q(x)}+\frac{B(x)}{R(x)}\text{.} \end{equation*}
