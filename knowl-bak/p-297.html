Clearly this is general. If we have an unspecified polynomial expanded about the number \(a\text{,}\) \(p(x) = a_0 + a_1(x-a) + a_2(x-a)^2 + \cdots + a_m(x-a)^n\) then clearly \(p(a) = a_1\) and \(\dfdx{p}{x}(a) = a_1\text{.}\) So it seems that in computing the coefficients \(a_0\) and \(a_1\) we have found the line tangent to \(p(x)\) at \(x=a\text{.}\) What do you suppose we will find when we compute \(a_2\text{?}\)
