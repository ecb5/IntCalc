    Section 2.3 The Opposite of the Product Rule -- Integration By Parts

    Substitution is a powerful tool, but it is not a panacea. For example, consider the integral
    \begin{equation*} \int x\cos(x)\dx{x}{}. \end{equation*}

    Comment.

    Actually, we should be more careful here. There is always a substitution that will work. The question is, can we find it?
    Try as you might you will find that no substitution you might try will work here. A common error that beginners make is to assume that that they can just integrate the two factors separately, giving
    \begin{equation} \int x\cos(x){}\dx{x}{}=\int x\dx{x}{}\int\cos(x)\dx{x}{}. \tag{2.4} \end{equation}
    Obviously equation (2.4) is not true. This doesn’t work for differentiation so there is no reason to think that it would work for integrals.
    Since the product rule is what we use to differentiate a product, then perhaps its analog would be of use here. If the product rule for differentiation is rearranged a bit it can be expressed as the Integration by Parts Formula:

    Note to self.

    I don’t think Problem 2.13 belongs here, or even that it is stated correctly.

    Problem 2.13.

    (a)

    Suppose \(f^\prime(x)\neq{}f(x)\) Show that if
    \begin{equation*} \left(f\cdot g\right)^\prime(x) = f^\prime(x)\cdot g^\prime(x) \end{equation*}
    then
    \begin{equation} g(x)=\exp\left({\int{\frac{1}{1-\frac{f(x)}{f^\prime(x)}}}\dx{x}{}}\right)\tag{2.5} \end{equation}
    (Don’t let this formula (2.5) frighten you. Recall that \(\exp(x)=e^x\text{.}\))

    (b)

    Apply equation (2.5) to the function \(f(x)=x^n\) and compare this to the results in Problem # 60 above.
    \begin{equation*} \int{}u\dx{v}{}=\int{} \left(\dx(uv) -v\dx{u}{}\right) = \int\dx(uv) -\int v\dx{u}{=uv-\int v\dx{u} }. \end{equation*}
    or
    \begin{equation} \int{}u\dx{v}{}=uv-\int v\dx{u}\text{.}\tag{2.6} \end{equation}

    Theorem 2.14. Integration by Parts Formula.

    \begin{equation} \dx(uv){}=u\dx{v}{}+v\dx{u}{.}\tag{2.7} \end{equation}
    Rearranging this a bit we have
    \begin{equation*} u\dx{v}{}=\dx(uv){}-v\dx{u}. \end{equation*}
    or
    \begin{equation*} \int{}u\dx{v}{}=\int{} \left(\dx(uv){}-v\dx{u}{}\right) = \int\dx(uv){}-\int v\dx{u}{=uv-\int v\dx{u}{}}. \end{equation*}
    The formula \(\int{}u\dx{v}{}=uv-\int v\dx{u}.\) is called the integration by parts formula and as you can see it is simply a rearrangement of the Product Rule formula. In this form it is exactly what we need to compute the previous integral. Specifically, if we let \(u=x\) and \(\dx{v}=\cos(x)\dx{x}\) , then \(\dx{u}=\dx{x}\) and \(v=\int{\cos(x){}\dx{x}}=\sin(x){}\) (don’t worry about the arbitrary constant here for the moment. We’ll look at it later.). Substituting these into our integration by parts formula,
    \begin{equation} \int{}u\dx{v}{}=uv-\int v\dx{u}\tag{2.8} \end{equation}
    we have
    \begin{equation*} \int x\cos(x)\dx{x} = x\sin(x)-\int\sin(x)\dx{x}=x\sin(x){}+\cos(x)+c{} \end{equation*}
    At this point, we should take note of a few things. First, notice that this was not a substitution; we did not end up with an integral with \(u\)’s or \(v\)’s in it. These were introduced just to keep track of the integration by parts formula.
    Second, notice that we needed to use the entire integrand. That is, the expression \(u\dx{v}\) is exactly equal to the the integrand, \(x\cos(x)\dx{x}\text{.}\) Nothing was left out.
    Second, notice that like a substitution, the integration by parts technique does not compute the integral. Rather, it replaces that integral with, hopefully, an easier integral. This means that as long as the integration by parts technique is applied correctly, it is not wrong; it just might not be helpful. For example, we could have let \(u=\cos(x){}\) and \(\dx{u}=x\dx{x}\text{.}\) This would give \(v=x^2/2\text{.}\) Substituting these into equation (2.8) we get
    \begin{equation*} \int x\cos(x)\dx{x}=\frac{x^2}{2}\cos(x)+\frac{1}{2}\int x^2\sin(x)\dx{x}. \end{equation*}
    Everything we’ve done is correct, but we have an integral that is worse than before. Practice is required to us this technique as well. Lots of practice.
    Lastly, notice that we suppressed the arbitrary constant when we computed \(v\text{.}\) We will see that this will not alter the result.

    Problem 2.15.

    (a)

    Redo the integration by parts on the integral \(\int x\cos(x)\dx{x}\) letting \(u=x\) and \(\dx{v}=\cos(x)\dx{x}\) as before. But suppose this time we take \(y=\sin(x)+a\) where \(a\) is an arbitrary constant. Show that we will obtain the same answer as before.

    (b)

    Show that in general, if we let \(\int\dx{v}=v+a\) in the integration by parts formula, then we will obtain the same result as before, namely \(\int u\dx{v}=uv-\int v\dx{u}\text{.}\)

    Problem 2.16.

    Notice that in Table 1.6 we had \(\int e^x\dx{x}=e^x+C \) but we had no analog for the integral of \(\ln(x)\text{.}\) Perform integral by parts on
    \begin{equation*} \int \ln(x) \end{equation*}
    using \(u=\ln(x){}\) and \(\dx{v}=\dx{x}\) to compute the integral.

    A Word of Advice.

    Rather than trying to memorize this integral, just remember to use integration by parts.
    As with differentiation techniques and formulas, techniques such as integration by parts and substitution are performed in concert, utilizing whatever technique that is appropriate at the time. As an example of this consider the next problem.

    Problem 2.17.

    Perform integration by parts and then substitution to compute \(\int \inverse\sin(x)\dx{x}\) and \(\int \inverse\tan(x)\dx{x}\text{.}\)
    Just as integration by parts can be used with other techniques, it can be used with itself multiple times.

    A Word of Advice.

    If you perform integration by parts twice, be sure not to switch the roles of “\(u\)” and “\(v\text{.}\)” This is not incorrect, but the second integration will simply undo the first integration and circle back to where you started. Try it and see.

    Problem 2.18.

    Compute
    \begin{equation*} \int x^2e^x\dx{x} \end{equation*}
    using integration by parts.
    Before we send you off to practice, we will look at one more “trick” one could do with integration by parts. We will also illustrate a way to keep track of \(u\text{,}\) \(\dx{v}\text{,}\) \(\dx{u}\text{,}\) and d \(v\text{.}\)

    Example 2.19.

    Consider
    \begin{equation*} \int e^x\sin(x)\dx{x}. \end{equation*}
    If we let \(u=e^x\) and \(\dx{v} =\sin(x)\dx{x}\text{,}\) then
    \begin{align*} u&=e^x &\dx{v}&=\sin(x)\dx{x}\\ \dx{u}&=e^x\dx{x}& v&=-\cos(x)\dx{x} \end{align*}
    so
    \begin{equation*} \int e^x\sin(x)\dx{x} = -e^x\cos(x){}+\int e^x\cos(x)\dx{x}. \end{equation*}
    We can apply integration by parts again, being careful not to switch the roles of \(u\) and \(v\text{.}\)
    \begin{align*} u&=e^x & \dx{v}&=\cos(x)\dx{x}\\ \dx{u}&=e^x\dx{x}& \sin(x){} \end{align*}
    so
    \begin{equation*} \int e^x\sin(x)\dx{x}=-e^x\cos(x)+\left[e^x\sin(x)-\int e^x\sin(x)\dx{x}\right]{}=-e^x\cos(x)+e^x\sin(x)-\int e^x\sin(x)\dx{x} \end{equation*}
    At first it looks like we went full circle and came back to where we started. But look again. If we let \(I=\int e^x\sin(x)\dx{x}\text{,}\) we see that this is now a simple algebra problem since
    \begin{equation*} I=-e^x\cos(x){+e^x\sin(x){-I}} \end{equation*}
    Solving for \(I\) we have
    \begin{equation*} I=\int e^x\sin(x)\dx{x}= \frac{e^x}{2}(\sin(x)-\cos(x))+c \end{equation*}
    Notice that we had to remember to add the arbitrary constant since there is no point in the calculation where it appears naturally .
