    Section 21.2 Accumulation: The Indefinite Integral

    In this section We will introduce the last variation on the integration notation we will use in this text.
    🔗
    We will need the following lemma about definite integrals, which we state without proof.
    🔗

    Lemma 21.2.0.1.

    If \(f(t)\) is integrable and \(a\) is in the domain of \(a\text{,}\) then
    \begin{equation*} \displaystyle \int_{t=a}^af(t)\dx{t} =0\text{.} \end{equation*}
    🔗
    🔗
    Eventually Lemma 21.2.0.1 will have to be proved rigorously. But you can see that it must be true since we start, and stop, integrating at the same point, \(t=a\text{.}\) So there is nothing to sum up, the integral must be zero.
    🔗
    To begin defining the indefinite integral observe that we can find the shaded area in Figure 21.2.0.2 by summing all differentials of the form \(f(t)\dx{t}\) from \(t=a\) to \(t=b\text{.}\) It follows that the definite integral \(\int_{t=a}^bf(t) \dx{t}\) represents that area.
    🔗
    Similarly, the integral \(\int_{t=a}^{x_1} f(t)\dx{t}\) represents the shaded area between \(t=a\) and \(t=x_1\text{,}\) \(\int_{t=a}^{x_2} f(t)\dx{t}\) represents the shaded area between \(t=a\) and \(t=x_2\text{,}\) \(\int_{t=a}^{x_3} f(t)\dx{t}\) represents the shaded area between \(t=a\) and \(t=x_3\text{,}\) and in general
    \begin{equation} \int_{t=a}^{x} f(t)\dx{t}\tag{21.10} \end{equation}
    represents the shaded area between \(t=a\) and \(t=x\text{.}\) An integral like the one in formula (21.10) is called an indefinite integral because it’s value depends on the location of \(x\text{.}\)
    🔗
    But we already have a term for this kind of dependency. When the value of one quantity depends on the value of another the first is said to be a function of the second. Thus we see that the indefinite integral in formula (21.10) represents a function of \(x\text{.}\)
    🔗
    Because we’ve built the function
    \begin{equation*} \int_{t=a}^xf(t)\dx{t} \text{ (note the presence of indices)} \end{equation*}
    by summing differentials in exactly the same way we built the most general antiderivative
    \begin{equation*} \int f(t)\dx{t} \text{ (note the absence of indices)}\text{,} \end{equation*}
    it should be clear that \(\int_{t=a}^xf(t)\dx{t}\) is one of the antiderivatives of \(f(t)\text{.}\)
    🔗

    Drill 21.2.0.3.

    Can you see which antiderivative it is? Take your best guess before reading on.
    🔗
    🔗
    The integral form of a function is cumbersome so we will only use it when there is a compelling need. Otherwise we’ll just choose a name for the function as usual. For instance we would use \(F(x)\) to name the function defined by formula (21.10) giving
    \begin{equation} F(x) = \int_{t=a}^{x} f(t)\dx{t} \text{.}\tag{21.11} \end{equation}
    This allows us to distinguish between \(F(x)\) and \(f(x)\text{,}\) while also indicating that they are related.
    🔗
    We only know one other fact about \(F(x)=\int_{t=a}^xf(t)\dx{t}\) aside from the fact that it is an antiderivative of \(f(x)\text{.}\) From Lemma 21.2.0.1 we know that
    \begin{equation*} F(a) = \int_{t=a}^af(t)\dx{t}=0 \text{.} \end{equation*}
    There can be only one antiderivative of \(f(x)\) with this property because any other would differ by some fixed amount, \(C\text{.}\)
    🔗
    So there it is. The function denoted by \(F(x)=\int_{t=a}^xf(t) \dx{t}\) is the unique antiderivative of \(f(x)\) with the property that when it is evaluated at \(x=a\) we get zero. A specific example will help clarify all of this.
    🔗

    Example 21.2.0.4. A Simple Indefinite Integral: Finding the Constant.

    Suppose we define \(F(x)\) as
    \begin{equation} F(x) = \int_{t=2}^x 3t\dx{t}\text{.}\tag{21.12} \end{equation}
    We know that \(\hat{F}(x) = \frac32x^2\) is an antiderivative of \(f(x)=3x\text{.}\) But \(\hat{F}(2)=6\neq{}0\text{,}\) so this is not antiderivative we are searching for.
    🔗
    However, because all of the antiderivatives of \(f(x)=3x\) differ by a constant it must be that
    \begin{equation*} F(x)=\hat{F}(x)+C=\frac32x^2+C \end{equation*}
    for some value of \(C\text{.}\) To find \(C\) we recall that \(F(2)=0\text{.}\) Thus
    \begin{equation*} F(2)=0{}=\eval{\frac32x^2+C}{x}{2} = \frac32\cdot 2^2+C\text{.} \end{equation*}
    Therefore \(C=-6 \) and thus
    \begin{equation} F(x)=\frac32x^2-6 \text{.}\tag{21.13} \end{equation}
    🔗
    🔗
    Because formula (21.13) is more familiar than formula (21.12) it is tempting to think of \(F(x)=\frac32x^2-6\) as the “correct” formula for \(F(x)\) and to think of \(\int_{t=2}^x 3t\dx{t}\) as an intermediate step in finding \(F(x)\text{.}\)
    🔗
    This is a fundamentally flawed view. In the same way that \(2\text{,}\) \(\sqrt{4}\text{,}\) \(\frac{6}{3} \text{,}\) or even \(\csc\left(\frac{\pi}{6}\right) \) are all alternative representations of the number two, formula (21.13) and formula (21.12) are two alternative ways to designate the function \(F(x)\text{.}\) They mean the same thing and which you choose to use will depend on your needs at the moment.
    🔗
    Of course in the absence of some specific need most people will use formula (21.13) because it is simpler to read and write, in the same way that we would use “\(2\)” unless there was some compelling reason not to.
    🔗

But when we solved the problems in Drill 8.1.0.2 in Section 8.1, for instance, we already knew that the derivative of \(x^2\) is \(2x\text{.}\) So all we had to do was satisfy the initial values. We now have the tools we need to solve an IVP without any prior knowledge of the solution.
    🔗

    Example 21.2.0.6. Some Initial Value Problems.

    (a)

    For example, if we want to solve the IVP
    \begin{gather*} \dfdx{y}{x}=\cos(x), \text{ } y(0)=-2. \end{gather*}
    we proceed as follows.
    🔗
    We need an antiderivative of \(\cos(x) \text{.}\) One antiderivative is
    \begin{equation*} y(x) = \int_{t=0}^x\cos(t)\dx{t} \end{equation*}
    but this is not the one we need since \(Y(0)=0\neq -2\text{.}\) It should be clear that
    \begin{equation*} y(x) = \int_{t=0}^x\cos(t)\dx{t}-2 \end{equation*}
    is the correct solution. But it is not very useful in this form because we have no way to evaluate \(y(x)\) for any value of \(x\) other than \(x=0\text{.}\)
    🔗
    Let’s look for something a little more useful. We know that \(\sin(x) \) is an antiderivative of \(\cos(x) \text{.}\) But \(\sin(x) \) does not satisfy our initial condition (since \(\sin(0)=0\neq-2 \)) so we need one of the other antiderivatives of \(\cos(x) \text{.}\) We don’t know which one so we’ll say that
    \begin{equation*} y(x) = \sin(x) +C \end{equation*}
    and look for the value of \(C\) that works. Thus the solution of our IVP must be
    \begin{equation*} y(x) = \sin(x)-2 \text{.} \end{equation*}
    🔗
    🔗

    (b)

    Of course we could have solved the IVP in part (a) by guessing just like we did before. Let’s look at an example that we can’t solve by guessing. Let’s solve the IVP
    \begin{equation*} \dfdx{y}{x}= \int \pi e^x\sin(\pi{}e^x) \dx{x}, \text{ } y(0)=7\text{.} \end{equation*}
    🔗
    First we need an antiderivative of \(\pi e^x\sin(\pi{}e^x)\text{.}\) Using the techniques we learned in Section 19.1 we see that one antiderivative is \(y(x)=-\cos(\pi e^x) \text{.}\) As usual this is the wrong one so we set \(y(x)=-\pi\cos(\pi{}e^x)+C\) and try to find the value of \(C\) that works. This is clearly
    \begin{equation*} y(x)=-\cos (\pi e^x)+6\text{.} \end{equation*}
    🔗
    🔗
    🔗

    DIGRESSION: Dummy Variables.

    You’ve probably noticed our use of both \(x\) and \(t\) that in equation (21.11). Do we really have two variables in play here as it appears we do or is something else happening? Give this some thought before you read on.
    🔗
    Don’t let this confuse you. In the formula \(F(x) = \int_{t=a}^xf(t)\dx{t} \) the variable for the function \(F\) is \(x\text{,}\) and the variable for the function \(f\) is \(t\text{.}\) The role of \(t\) is to define \(f(t)\) so we can, for instance, display its graph as in Figure 21.2.0.2. Notice that there is no \(x\)– axis in the figure. The graph of \(f(t)\) is shown on the \(y-t\) plane.
    🔗
    The role of \(x\) is the same as the role of \(a\text{,}\) or \(b\text{.}\) It locates a position on the \(t\)–axis. But since \(F(x)\) depends on the value of \(x\) it is the variable for \(F\text{.}\)
    🔗
    In fact, in a real sense the variable \(t\) isn’t really present. That it appears to be is an artifact of defining the function \(F(x)\) via an integral. When we do this we must have another function to integrate. This forces us to use a different variable name to keep things orderly.
    🔗
    To see this consider the function
    \begin{equation*} F(x) = \int_{t=1}^xt\dx{t}\text{.} \end{equation*}
    We have
    \begin{equation*} F(x) = \eval{\frac{t^2}{3}}{t}{1}^x = \frac{x^2}{2}-\frac{1}{2} \end{equation*}
    or
    \begin{equation*} F(x) = \frac{x^2}{2}-\frac{1}{2} \text{.} \end{equation*}
    Notice that there is no \(x\) in the last formula. For better or worse the integration variable (\(t\) in this case) is called a dummy variable.
    🔗
    END OF DIGRESSION

    We saw in the previous section that when computing a definite integral it doesn’t matter which antiderivative of \(\int f(x)\dx{x}\) we choose since the constant will subtract out anyway.
    🔗
    However, there are situations where the particular antiderivative used is very important.
    🔗

    Example 21.2.0.8.

    For example consider the IVP
    \begin{align} \dfdx{y}{x} = 3x^{\frac52 } +x, \amp{}\amp{} y(0)=2\tag{21.14} \end{align}
    To solve this observe that
    \begin{equation*} \int\dx{y} = \int(3x^{\frac52 } +x)\dx{x}\text{.} \end{equation*}
    Finding the most general antiderivative of each integral gives
    \begin{equation} y(x) +C_1=\frac67 x^{\frac72} + \frac12 x^2 +C_2\text{.}\tag{21.15} \end{equation}
    Strictly speaking (21.15) relates two multifunctions but suppose \(y(x)\) is the particular solution of our IVP that we seek. Combining the constants we have
    \begin{equation*} y(x) =\frac67 x^{\frac72} + \frac12 x^2 + \underbrace{C_2-C_1}_{=C}\text{.} \end{equation*}
    We need to find the value of \(C=C_2-C_1\) which satisfies our IVP.
    🔗
    From the initial condition we know that
    \begin{equation*} y(0) = C=2 \end{equation*}
    so
    \begin{equation*} y(x) =\frac67 x^{\frac72} + \frac12 x^2 + 2 \end{equation*}
    🔗

    Example 21.2.0.10. Population Dynamics, Redux.

    For instance in problem 8.6.0.3.a you were asked to verify that the solution of the IVP
    \begin{align} \dfdx{P}{t}= 0.3P,\amp{}\amp{} P(0)=5.\tag{21.16} \end{align}
    is \(P(t)=5e^{0.3t}\) but we gave no indication how we arrived at the solution in the first place. Since \(\dfdx{P}{t}= 0.3P\) is a separable differential equation we now have the tools we need to derive the solution ourselves.
    🔗
    Since this is separable we see that
    \begin{equation} \int\frac{1}{P}\dx{P}=\int0.3\dx{t}\text{.}\tag{21.17} \end{equation}
    Computing the most general antiderivative of both sides as in Example 21.2.0.8 we have
    \begin{equation*} \ln\left(\abs{P}\right) = 0.3t+C\text{.} \end{equation*}
    🔗

    Aside: Mathematical Notation.

    The absolute value bars in
    \begin{equation*} \ln\left(\abs{P}\right) = 0.3t+C \end{equation*}
    are correct, but for this problem they turn out to be irrelevant. So we will ignore them from here.
    🔗
    From our initial condition we have
    \begin{equation*} \ln(5) = C \end{equation*}
    so
    \begin{equation*} \ln(P)= 0.3t+\ln(5) \text{.} \end{equation*}
    But we want to find \(P\) not \(\ln(P) \text{.}\) Exponentiating both sides gives
    \begin{equation*} e^{\ln\left(P\right)} = e^{0.3t+\ln(5)} = e^{0.3t}\cdot e^{\ln(5)} = 5e^{0.3t} \end{equation*}
    or
    \begin{equation*} P(t) = 5e^{0.3t}\text{.} \end{equation*}
    🔗

    The Indefinite Integral as an Accumulation Function.

    We have consistently described the definite integral
    \begin{equation*} F(x)=\int_{t=a}^{b}f(t) \dx{t} \end{equation*}
    as summing the differentials \(f(t)\dx{t}\) “starting at \(x=a\)” and “ending at \(x\text{.}\)” This suggests that we can think of the function
    \begin{equation*} F(x) = \int_{t=a}^x f(t)\dx{t} \end{equation*}
    as an accumulation function. It starts at \(t=a\) and proceeds to \(t=b\text{,}\) gathering in and adding the differentials \(f(t)\dx{t}\) along the way. Thus, for example the function
    \begin{equation*} F(x) = \int_{t=2}^xt \dx{t} \end{equation*}
    in Example 21.2.0.4 begins accumulating differentials of the form \(3t\dx{t}\) at \(t=2\) and continues to add them up until it reaches the point \(t=x\text{.}\) The value of \(F(x)\) is the sum of all of those differentials.
    🔗
    Another formulation of this particular function is
    \begin{equation*} F(x) = \frac32 t^2-6\text{.} \end{equation*}
    Since this is the same function, just reformulated, it is still an accumulation function, but the algebraic notation doesn’t reflect this as clearly as the integral notation.
    🔗
    Thinking of the indefinite integral this way can help us understand its properties a little better. For instance, we saw in Theorem 21.1.2.5 that
    \begin{equation*} \int_a^bt\dx{t} = \eval{t}{\frac{t^2}{2}}{a}^b\text{.} \end{equation*}
    We know how to compute this now but let’s instead consider how the notion of an accumulation function sheds light onto the meaning of the computation.
    🔗
    First let’s define the function \(F(x)= \int_{t=0}^x t\dx{t}\) which accumulates the area of the red region in the following sketch from \(t=0\) over to \(t=x\text{.}\)
    🔗
    If \(x=10\) then \(F(10)\) will be the area of the red triangle and similarly, \(F(15)\) will be the area of the larger red and blue triangle. But how could we represent the area of the blue region alone?
    🔗
    Clearly
    \begin{equation*} \text{Area}\left( \text{Blue}\right) =\text{Area}\left(\text{Red}+\text{Blue}\right)-\text{Area}\left( \text{Red}\right) \end{equation*}
    or
    \begin{equation*} \int_{t=10}^{15}f(t) \dx{t}=F(15)-F(10) \end{equation*}
    which is exactly what we would have gotten had we applied Theorem 21.1.2.5 to this problem.
    🔗
    Our point here is that it is easier to see why Theorem 21.1.2.5 must be true by reasoning from Figure 21.2.0.2. It is useful to think of integrals, both definite and indefinite, in terms of an accumulation function.
