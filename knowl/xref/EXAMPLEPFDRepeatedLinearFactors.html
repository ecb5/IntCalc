Example 19.2.2.7. The PFD With Repeated Linear Factors.
Let’s finish up the decomposition of \(\frac{x}{(x-1)^2(x+2)^2}\) from part (b) of Problem 19.2.2.6. From the HCUM You should have found that \(B=\frac19\) and \(D=\frac{-2}{9}\) so that
\begin{equation*} \displaystyle \frac{x}{(x-1)^2(x+2)^2}= \frac{A}{x-1} + \frac{\frac19}{(x-1)^2} +\frac{C}{x+2} -\frac{\frac29}{(x+2)^2} \end{equation*}
When we reached this point in Example 19.2.2.4 our next step was to simply choose a value for \(x\text{.}\) We could do that here as well but we’d have to choose two values for \(x\) thereby generating a pair of equations to solve and we’ve already seen that there are more efficient methods for computing the PFD than solving a system of equations.
Instead we’ll isolate all of the terms with our unknowns on one side of the equation, thus:
\begin{equation} \displaystyle \frac{x}{(x-1)^2(x+2)^2} - \frac{\frac19}{(x-1)^2} +\frac{\frac29}{(x+2)^2} = \frac{A}{x-1} +\frac{C}{x+2}\tag{19.46} \end{equation}
It is clear that the right side of equation (19.46) is the PFD of some rational function whose denominator is \((x-1)(x+2)\text{.}\) That is, if we were asked to begin the process of computing the PFD of
\begin{equation*} \frac{P(x)}{(x-1)(x+2)} \end{equation*}
for some polynomial \(P(x)\) then our first step would be to write down the right side of equation (19.46). Therefore it must be that
\begin{equation} \displaystyle \frac{x}{(x-1)^2(x+2)^2} - \frac{\frac19}{(x-1)^2} +\frac{\frac29}{(x+2)^2} = \frac{P(x)}{(x-1)(x+2)}\tag{19.47} \end{equation}
for some polynomial \(P(x)\text{,}\) and if we can find that polynomial we can compute \(A\) and \(C\) by the HCUM.
We can find \(P(x)\) by essentially the same method we used to find the constant \(A\) in Example 19.2.2.4. First we clear the fraction on the right
\begin{equation} \frac{x}{(x-1)(x+2)} - \frac19 \left( \frac{x+2}{x-1} \right) +\frac29 \left( \frac{x-1}{x+2} \right) = P(x)\tag{19.48} \end{equation}
Since the left side is guaranteed to reduce to the polynomial it is tempting to find a common denominator and add the three fractions on the left together. Since we have to end up with the polynomial \(P(x)\) it must be that once the addition is done there will be a factor in the top that will cancel with the common denominator.
That will work. Give it a try if you want. Good luck.
But we’re trying to find the most efficient method we can so instead we’re going to make judicious use of the knowledge that we must end up with a polynomial.
First think about the expression in the second term on the left: \(\frac{x+2}{x-1}\text{.}\) If we perform this division we get
\begin{equation*} \frac{x+2}{x-1} = 1 +\frac{3}{x-1}\text{.} \end{equation*}
Similarly if we computing the division in the third term we get \(\frac{x-1}{x+2}= 1-\frac{3}{x+2}\text{.}\) Putting these back into equation (19.48) we have
\begin{equation*} \frac{x}{(x-1)(x+2)} - \frac19 \left( 1 +\frac{3}{x-1} \right) +\frac29 \left( 1-\frac{3}{x+2} \right) = P(x) \end{equation*}
or
\begin{equation*} \frac{x}{(x-1)(x+2)} - \frac19 -\frac{\frac13}{x-1} +\frac29 -\frac{\frac23}{x+2} = P(x) \text{.} \end{equation*}
Since \(P(x)\) must be a polynomial it follows that \(\frac{x}{(x-1)(x-2)}-\frac{\frac13 }{x-1}-\frac{\frac23 }{x+2}\) (everything that isn’t a polynomial) must sum to zero. So we can ignore them and conclude that our polynomial is given by the other terms. So
\begin{equation*} P(x)=-\frac19 + \frac29 =\frac19\text{.} \end{equation*}
To finish the decomposition we observe that
\begin{equation*} \frac{\frac19 }{(x-1)(x+2)} = \frac{A}{x-1}+\frac{C}{x+2} \end{equation*}
and from the HCUM we see that \(A=\frac{1}{27} \) and \(C=-\frac19\text{.}\)
Having found all of the constants we conclude that
\begin{equation*} \frac{x}{(x-1)^2(x+2)^2}= \frac{\frac{1}{27}}{x-1} + \frac{\frac19 }{(x-1)^2} -\frac{\frac19}{x+2} -\frac{\frac29 }{(x+2)^2} \text{.} \end{equation*}
