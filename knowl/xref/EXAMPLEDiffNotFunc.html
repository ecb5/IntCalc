Example 1.2. We Integrate Differentials, Not Functions.

It is common to speak informally of “the integral of the function \(f(x)=2x\)” but this is not the correct usage. When we write \(\int 2x\dx{x}\) we are summing the differential  4  \(2x\dx{x} (=\dx{y})\text{,}\) not the function \(2x\text{.}\) It is important to keep this basic fact in mind as we go forward because it will save us much confusion later.
If \(f(x)\) is some function the expression
\begin{equation*} \int f(x) \end{equation*}
is meaningless, despite the fact that nearly everyone occasionally abuses the notation in this way. The integral sign is intended to express the summation of differentials. Any other usage is meaningless, no matter how convenient it might be.
Suppose that \(y=x^2\) then \(\dx{y} =2x\dx{x}\text{.}\) Then \(\int 2x\dx{x}\) makes sense because the product \(2x\dx{x}=\dx{y}\) is a differential. But \(\int 2x\) is nonsense because \(2x\) alone is not a differential.
In particular, \(\int 2x\) is meaningless. However if \(y=x^2\) then what is probably meant is
\begin{equation} \int 2x\dx{x} = \int \dx{y}\tag{1.2} \end{equation}
which is equal to \(y\) since \(\int \dx{y}\) means “add up the differentials of \(y\text{.}\)”
