Example 19.1.4.22.

What happens if you don’t have an odd power of sine or cosine? For example, suppose you have \(\int\sin^2(\theta)\cos^2(\theta)\dx{\theta}\text{.}\)
🔗
Aside: Note to self.
We may want to consider moving Polynomials in secant and tangent to just above this example.
🔗
Saving a sine or cosine would lead to an integrand with a square root, which we are trying to avoid, if we can help it. In Problem 19.1.4.7 we saw that one of the Half–Angle formulas from Trigonometry was helpful. In fact, the Half-Angle formlas:
\begin{equation*} \sin^2(\theta) = \frac12\left(1-\cos(2\theta)\right) \text{ and } \cos^2(\theta)=\frac12\left(1+\cos(2\theta)\right) \end{equation*}
are just the tool we need when there is no way to “save” a sine or cosine. Using these identities in the integral
\begin{equation*} \int\sin^2( \theta )\cos^2 (\theta )\dx{x} \end{equation*}
we see that
\begin{align*} \int\sin^2(\theta)\cos^2(\theta)\dx{\theta}\amp =\frac14\int(1-\cos(2\theta))(1+\cos(2\theta))\dx{\theta}\\ \amp =\frac14\int (1-\cos^2(2\theta))\dx{\theta}\\ \amp =\frac{1}{4}\int\sin^2(2\theta)\dx{\theta} \end{align*}
which doesn’t seem to help much until we notice that we can use the Half-Angle formula again. This gives
\begin{equation*} \frac{1}{4}\int\sin^2(2\theta)\dx{\theta} = \frac18\int(1-\cos(4\theta))\dx{\theta} = \frac18\left(\theta-\frac14\sin(4\theta)\right)+C \end{equation*}
