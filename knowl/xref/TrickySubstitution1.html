Problem 2.9.

On the other hand we now know that \(\int\tan(x)\dx{x}=\ln(\sec(x)+c\) and we can use this knowledge. Suppose we need to compute \(\int\frac{u}{1+u^2}\dx{u}\) or \(\int\frac{u}{1-u^2}\dx{u}\text{.}\) The same substitution we used to transform these into the integral \(\int\tan(x)\dx{x}\) will work in reverse.
(a)
Use the substitution \(u=\tan(x)\) to show that:
\begin{equation*} \int\frac{u}{1+u^2}\dx{u} = \ln\left(\sec\left(\inverse\tan(u)\right)\right)+c. \end{equation*}
Now use your Algebra and Trigonometry skills to show that \(\sec\left(\inverse\tan(u)\right) = \sqrt{1+u^2}\) so that
\begin{equation*} \int\frac{u}{1+u^2}\dx{u} = \frac12\ln\left(1+u^2\right)+c\text{.} \end{equation*}
(b)
Use the substitution \(u=\sin(x)\) to show that
\begin{equation*} \int\frac{u}{1-u^2}\dx{u}= \ln\left(\sec\left(\inverse\sin(u)\right)\right)+c. \end{equation*}
Now use your Algebra and Trigonometry skills to show that \(\sec\left(\inverse\sin(u)\right) = \sqrt{1-u^2}\) so that
\begin{equation*} \int\frac{u}{1-u^2}\dx{u} = -\frac12\ln\left(1-u^2\right)+c\text{.} \end{equation*}
