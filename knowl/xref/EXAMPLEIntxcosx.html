Example 19.3.0.3.

To integrate \(\int x\cos (x)\dx{x}\) by parts we set
\begin{align*} u=x\amp{}\amp{}\text{ and } \amp{}\amp{} \dx{v}=\cos(x)\dx{x}\\ \end{align*}
so that
\begin{align*} \dx{u}=\dx{x} \amp{}\amp{}\text{ and } \amp{}\amp{} v=\int\cos(x)\dx{x} = \sin(x). \end{align*}
Making these substitutions gives
\begin{align*} \int x\cos(x)\dx{x} = \int u\dx{v}\amp{}=uv-\int v\dx{u}\\ \amp{}=x\sin(x) -\int \sin (x)\dx{x}.\\ \amp{}=x\sin(x) + \cos (x). \end{align*}
and, finally
\begin{equation} \int x\cos(x)\dx{x} = x\sin(x) + \cos (x) +C\tag{19.68} \end{equation}
