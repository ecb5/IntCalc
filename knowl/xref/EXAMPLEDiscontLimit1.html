Example 14.1.1.4.

Suppose
\begin{equation*} f(x)= \begin{cases} 3x \amp \text{if } x\neq2\\ 10 \amp \text{if } x=2 \end{cases}. \end{equation*}
Then
\begin{equation*} \limit{x}{2}{f(x)}= 6. \end{equation*}
In particular the limit is not \(10\text{.}\) Here is how we would evaluate this limit using the tools we currently have at our disposal.
