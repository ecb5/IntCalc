Problem 19.1.4.6.

Complete Example 19.1.4.5 by using the substitution \(x=\tan (\theta )\) to show that
\begin{equation*} \frac12 \theta + \frac14 \sin (2\theta ) = \frac12\left(\inverse\tan(x) +\frac{x}{1+x^2}\right)\text{.} \end{equation*}
Hint.
You will need the Double–Angle formula for the sine function from Trigonometry.
