Problem 17.2.2.13.
Suppose \(f(x)\) is defined near \(x=a\text{.}\) Show that
\begin{equation*} \limit{x}{a}{f(x)}=\infty \end{equation*}
if and only if for any real number \(B>0\text{,}\) there is a \(\delta >0\) such that \(f\left(x\right)\gt B\) whenever \(a-\delta \lt x\lt a+\delta \) and \(x\neq a\text{.}\)
