Task 15.1.0.5.a.

The Absolute Value function is not differentiable at \(x=0\) but it is differentiable at every other value of \(x\text{.}\) Let \(f(x)=\abs{x}\) and use Definition 12.2.0.3 to show that:
  1. If \(x\gt0\) then \(f^\prime(x)=1\text{,}\)
  2. If \(x\lt0\) then \(f^\prime(x)=-1\text{.}\)
