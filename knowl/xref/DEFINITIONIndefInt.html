Definition 21.1.2.2. Indefinite Integral.

Given the function \(f(x)\) the formula
\begin{equation*} \int f(x)\dx{x} \end{equation*}
always represents the multifunction whose derivative is \(f(x)\text{.}\) Such an expression is called an indefinite integral.
In particular, if \(F(x)\) is any antiderivative of \(f(x)\) Then
\begin{equation} \int f(x)\dx{x} = F(x) +C\text{,}\tag{21.5} \end{equation}
where, as before, \(C\) represents all possible constants.
