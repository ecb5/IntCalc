Problem 17.2.2.9.

Let \(c\) be a real number in the interval \((a,b)\) and suppose \(f(x)\) is defined on some interval \((a,b)\text{.}\) Use Definition 17.2.2.2 and Definition 17.2.2.8 to prove that
\begin{equation*} \rlimit{x}{a}{f(x) }=-\infty \end{equation*}
if and only if
\begin{equation*} \rlimit{x}{a}{\left(-f\left(x\right)\right) }=\infty\text{.} \end{equation*}
