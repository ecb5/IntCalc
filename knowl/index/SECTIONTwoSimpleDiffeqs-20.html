Functions, Mulitifunctions and Integral Notation.

But first we need to take another look at the function/multifunction issuse we raised earlier. As we saw in Example 3.1.2.2 the expression
\begin{equation*} \int 2t\dx{t} \end{equation*}
represents the most general antiderivative of \(f(t)=2t\text{.}\) In this simple case we can see that this is the multifunction
\begin{equation*} f(t)=t^2+C \text{.} \end{equation*}
Suppose we need to solve the Initial Value Problem
\begin{align*} \dfdx{y}{t}=2t\amp{}\amp{} \text{ and }\amp{}\amp{} y(0)=5. \end{align*}
To accomplish this we need to solve the differential equation \(\dfdx{y}{t}=2t \) to find the multifunction
\begin{equation*} y(t)= \int{}\dx{y}=\int 2t\dx{t}\text{,} \end{equation*}
and then find the particular function represented by \(\int 2t\dx{t}\) which satisfies the initial condition \(y(0)=4\text{.}\)

Drill 4.1.0.6.

You may already see that \(y(t)\) must be \(y(t)=t^2+5\text{.}\) If so, excellent! If not, take a moment to convince yourself that this must be the case.
In either case, bear with us while we explain the details .
Since \(y(t)=\int 2t\dx{t} \) and \(y(0)=5 \) we know that the particular function we seek satisfies
\begin{equation*} y(0)= \eval{\int 2t\dx{t}}{t}{0} =5 \text{.} \end{equation*}
This is the best we can do if we know nothing else about the particular function. But in this case we also know that \(y(t)=t^2+C\) for some value of \(C\text{.}\) So we see that
\begin{equation} y(0)= \eval{\int 2t\dx{t}}{t}{0}= \eval{t^2+C}{t}{0}=0^2+C =5 \text{,}\tag{4.9} \end{equation}
from which it follows that
\begin{equation*} y(t)=t^2+5\text{.} \end{equation*}
You will become very familiar with this process as we proceed. As Calculus developed the integral notation evolved from the simple “S” of Leibniz into a sophisticated and esoteric bit of notation, which takes some effort to get comfortable with, so we will introduce the changes incrementally.
For example, while our evaluation notation \(\eval{\int 2t\dx{t}}{t}{0}\) is clear it is also rather clumsy in the context of integration. So we will modify it slightly in this context as follows. Given \(f(t)\text{,}\) it’s indefinite integral at \(t=a\) is given by
\begin{equation} \eval{\int f(t)\dx{t}}{t}{a} = \int_{t=a} f(t)\dx{t} \text{.}\tag{4.10} \end{equation}
This change appears hardly worth the effort at the moment but as we mentioned the integral notation has evolved quite a bit over time. Formula (4.10) is just the first modification we’ll see.
Using our new notation we see that equation (4.9) can be expressed as
\begin{equation*} y(0)=\int_{t=0}2t\dx{t} = 0^2+C=5\text{.} \end{equation*}
The convenience of this notation may not be clear to you yet but as we proceed you will see that it is very helpful.
