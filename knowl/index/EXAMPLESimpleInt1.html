Example 1.1.2.2. A Simple Integration.

For example, can you find a formula for this integral:
\begin{equation*} \int2t\dx{t}\text{?} \end{equation*}
Clearly \(f(t)=2t\) so from Definition 1.1.2.1 we see that \(F(t) = \frac{t^2}{2}\) is one antiderivative, and we can capture all of the antiderivatives by adding an arbitrary constant, so
\begin{equation*} \int 2t\dx{t} = \frac{t^2}{2} + C \end{equation*}
is correct.
This shows that the expression \(\int f(t)\dx{t} \) represents a multifunction much like the “\(\text{arc}\)” functions we encountered in [cross-reference to target(s) "CHAPTERcalc-trig" missing or not unique]. However, none of the complicated domain and range difficulties that caused so many headaches in [cross-reference to target(s) "SECTIONinverse-tan-and-cot" missing or not unique] will be in play here. Since the multifunctions arising from integration are relatively simple, we will dispense with the formal language and call
\begin{equation*} F(t) = \int 2x\dx{x} = \frac{t^2}{2}+C \end{equation*}
the most general antiderivative of \(f(t) = t^2\text{.}\) There will be times when we will want to choose only one of the functions represented by \(\int f(t)\dx{t}\text{.}\) This will consist entirely of choosing the value of \(C\) that works for the problem in front of us.
