Functions, Mulitifunctions and Integral Notation.

Before proceeding we will take a moment to look again at the function/multifunction issue we raised earlier. As we saw in Example 17.1.2.2 the expression
\begin{equation*} \int 2t\dx{t} \end{equation*}
represents the most general antiderivative (a multifunction)_of \(f(t)=2t\text{.}\) In this simple case we can see that this is the multifunction also has the form
\begin{equation*} f(t)=t^2+C \text{.} \end{equation*}
Suppose we need to solve the Initial Value Problem
\begin{align} \dfdx{y}{t}=2t\amp{}\amp{} \text{ and }\amp{}\amp{} y(0)=5.\tag{18.9} \end{align}
To accomplish this we need to solve the differential equation \(\dfdx{y}{t}=2t \) to find the multifunction
\begin{equation*} y(t)= \int{}\dx{y}=\int 2t\dx{t}\text{,} \end{equation*}
and then find the particular function represented by \(\int 2t\dx{t}\) which satisfies the initial condition \(y(0)=4\text{.}\)

Comment.

You may already see a simpler representation of \(y(t)\text{.}\) If so, excellent! If not, take a moment to convince yourself that you are right. Otherwise, bear with us while we explain the details.
Since \(y(t)=\int 2t\dx{t} \) and \(y(0)=5 \) we know that the particular function we seek satisfies
\begin{equation*} y(0)= \eval{\int 2t\dx{t}}{t}{0} =5 \text{.} \end{equation*}
This is the best we can do if we know nothing else about this particular function. But from Example 17.1.2.2 we see that \(y(t)\) also has the form
\begin{equation*} y(t)=t^2+C \end{equation*}
for some value of \(C\text{.}\) So we see that
\begin{equation} y(0)= \eval{\int 2t\dx{t}}{t}{0}= \eval{t^2+C}{t}{0}=0^2+C =5\text{.}\tag{18.10} \end{equation}
It follows that the solution of IVP (18.9) is the function \(y(t)\) selected from our multifunction \(\int 2t\dx{t} = t^2+C\)/. Specifically,
\begin{equation*} y(t)=t^2+5\text{.} \end{equation*}
While our evaluation notation is clear, in this context the expression \(\eval{\int 2t\dx{t}}{t}{0}\) is also rather clumsy. So we will modify it slightly in this context as follows:
\begin{equation*} y(0)= \eval{\int 2t\dx{t}}{t}{0}=\int^{t=0}2t\dx{t} \text{.} \end{equation*}
More generally, the indefinite integral of \(f(t)\text{,}\) at \(t=a\) is given by
\begin{equation} \eval{\int f(t)\dx{t}}{t}{a} = \int^{t=a} f(t)\dx{t} \text{.}\tag{18.11} \end{equation}
Our point here is that as Calculus developed from the ideas of Leibniz and Newton into the powerful tool is has become, the integral notation evolved from the simple “S” of Leibniz into a sophisticated and rather esoteric bit of notation. It takes some time and effort to get comfortable with it, so we will introduce the changes incrementally.
Using our new notation we see that equation (18.10) can be expressed as
\begin{equation*} y(0)=\int^{t=0}2t\dx{t} = 0^2+C=5\text{.} \end{equation*}
The convenience of this notation may not be clear to you yet but as we proceed you will see that it is very helpful, particularly in the context of Definite Integration which we will return to in Chapter 20.
