Next suppose we need to solve the Initial Value Problem
\begin{align} \dfdx{y}{t}=2t\amp{}\amp{} \text{ and }\amp{}\amp{} y(0)=5.\tag{4.4} \end{align}
To accomplish this we need to solve the differential equation \(\dfdx{y}{t}=2t \) to find the multifunction
\begin{equation*} y(t)= \int{}\dx{y}=\int 2t\dx{t} \text{,} \end{equation*}
and then find the particular function which satisfies the initial condition \(y(0)=4\text{.}\)
