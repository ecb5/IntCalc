Problem 18.1.2.3.

Show that the following statements are true. Assume \(C\) is an arbitrary constant. (We have not shown you how to compute any of these integrals. But you know how to differentiate. Read Definition 18.1.2.1 again carefully.)
(a)
\(\displaystyle\int 2te^{t^2}\dx{t} = e^{t^2} +C\)
(b)
\(\displaystyle\int te^t\dx{t} = (t-1)e^t+C\)
(c)
\(\displaystyle\int \sin(x)\cos(x)\dx{x} =\frac12\sin^2(x)+C\)
Hint.
Notice that we’ve used \(x\) instead of \(t\) for our variable. Does that matter? Explain.
(d)
\(\displaystyle\int \sin(\theta )\cos(\theta )\dx{\theta } = -\frac12\cos^2(\theta )+C\)
(e)
\(\displaystyle\int u\cos(u) \dx{u} = u\sin(u)+\cos(u)+C\)
(f)
\(\displaystyle\int x\ln(x) \dx{x} = x(\ln{x}-1)+C\)
