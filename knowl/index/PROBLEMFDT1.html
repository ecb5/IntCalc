Problem 9.5.3.9.

(a)
For the objective function
\begin{equation*} A(x) = (x-2)^2 + (x^2-3)^2 \end{equation*}
choose a point in each interval and confirm that the following statements are true.
  1. On the (approximate) interval \((-\infty, -1.32)\text{:}\) \(A^\prime(x) \lt0\)
  2. On the (approximate) interval \((-1.32, -0.43)\text{:}\) \(A^\prime(x) \gt0\)
  3. On the (approximate) interval \((-0.43, 1.75)\text{:}\) \(A^\prime(x) \lt0\)
  4. On the (approximate) interval \((1.75, \infty)\text{:}\) \(A^\prime(x) \gt0\)
(b)
Use the First Derivative Test and the data in part 9.5.3.9.a to show that there is a local minimum at \(x\approx-1.32\) a local maximum at \(x\approx-0.43\) and a local minimum at \(x\approx1.75\text{.}\)
(c)
Is the data in part 9.5.3.9.a sufficient to support the conclusion that there is a global minimum at \(x\approx1.75\text{?}\) Explain why not.
(d)
What additional data do we need to conclude that there is a global minimum at \(x\approx1.75\text{?}\)
