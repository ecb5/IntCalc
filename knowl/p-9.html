To see why, notice that if \(\dx{f}=\dx{g} \) then \(\dx(f-g)=\dx{f}-\dx{g}=0\text{.}\) Dividing by \(\dx{x}\) so we can think of this as a slope, we have \(\frac{\dx{(f-g)}}{\dx{x}}=0\text{.}\) What kind of curve would have a slope which is constantly \(0\text{?}\) A moment’s thought will suggest that this curve must be a horizontal line for if not, then somewhere it should have a nonzero slope. Such an argument can be made rigorous, but we will not fuss over that right now.
