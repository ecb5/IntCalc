For example, consider the integral
\begin{equation*} \int x^2\sqrt{x^3+1}\dx{x} \end{equation*}
We could let \(u = x\) so that \(\dx{u} = 2x \dx{x}\) . This would lead to
\begin{equation*} \int x^2\sqrt{x^3+1}\dx{x}= \int u\sqrt{u^{3/2}+1}\left(\frac{\dx{u}}{2x}\right) = \frac12\int\frac{u\sqrt{u^{3/2}+1}}{x}\dx{u}. \end{equation*}
Notice that we can bring the \(1/2\) outside the integral because it is a constant, but since \(\frac{1}{x}\) is not a constant (it depends on \(u\)) we cannot do the same with with it. We could rewrite \(x = \sqrt{u}\) , so that
\begin{align*} \int x^2\sqrt{x^3+1}\dx{x} = \frac12\int\frac{u\sqrt{u^{3/2}}}{\sqrt{u}}\dx{u} \amp= \frac12 \int\sqrt{u}\sqrt{x^{3/2}+1}\dx{u}\\ \amp=\frac12\int\sqrt{u^{5/2}+u}\dx{u}. \end{align*}
This is all correct, but leads to an integral far worse than the original, especially if you notice that \(x\) is \(3\) “almost” the derivative of \(x + 1\text{.}\) It is off by a factor of \(3\text{,}\) but as we’ve seen, constants can be brought outside of the integral, so this is not really a problem. So let’s make \(y = x + 1\) , so that \(\dx{y} = 3x \dx{x}\text{.}\) Thus
\begin{align*} \int x^2\sqrt{x^3+1}\dx{x}\amp=\int x^2\sqrt{y}\left(\frac{\dx{y}}{3x^2}\right)\\ \amp=\frac13\int y^{1/2}\dx{y}\\ \amp=\frac13\left(\frac{y^{3/2}}{3/2}\right)+c\\ \amp=\frac29\left(x^3+1\right)^{3/2}+c \end{align*}
