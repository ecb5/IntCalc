To find the maximum height, we set \(v=0\text{.}\) So we see that \(0=-9.8t+19.6 \text{.}\) Solving for \(t\) gives \(t=2 \text{ seconds}\text{.}\) Thus the ball’s maximum height is
\begin{equation*} y(2)=-4.9(2^2)+19.6(2)+2. \end{equation*}
So the ball will reach a height of \(= 21.6 \text{ meters}\text{.}\)
