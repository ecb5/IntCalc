Example 2.20.

Consider
\begin{equation*} \int e^x\sin(x)\dx{x}. \end{equation*}
If we let \(u=e^x\) and \(\dx{v} =\sin(x)\dx{x}\text{,}\) then
\begin{align*} u\amp=e^x \amp\dx{v}\amp=\sin(x)\dx{x}\\ \dx{u}\amp=e^x\dx{x}\amp v\amp=-\cos(x)\dx{x} \end{align*}
so
\begin{equation*} \int e^x\sin(x)\dx{x} = -e^x\cos(x){}+\int e^x\cos(x)\dx{x}. \end{equation*}
We can apply integration by parts again, being careful not to switch the roles of \(u\) and \(v\text{.}\)
\begin{align*} u\amp=e^x \amp \dx{v}\amp=\cos(x)\dx{x}\\ \dx{u}\amp=e^x\dx{x}\amp \sin(x){} \end{align*}
so
\begin{equation*} \int e^x\sin(x)\dx{x}=-e^x\cos(x)+\left[e^x\sin(x)-\int e^x\sin(x)\dx{x}\right]{}=-e^x\cos(x)+e^x\sin(x)-\int e^x\sin(x)\dx{x} \end{equation*}
At first it looks like we went full circle and came back to where we started. But look again. If we let \(I=\int e^x\sin(x)\dx{x}\text{,}\) we see that this is now a simple algebra problem since
\begin{equation*} I=-e^x\cos(x){+e^x\sin(x){-I}} \end{equation*}
Solving for \(I\) we have
\begin{equation*} I=\int e^x\sin(x)\dx{x}= \frac{e^x}{2}(\sin(x)-\cos(x))+c \end{equation*}
Notice that we had to remember to add the arbitrary constant since there is no point in the calculation where it appears naturally .
