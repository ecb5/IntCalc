Problem 19.1.4.16.

If we need to compute integral
\begin{equation*} \int\frac{x}{\left(x^2-1\right)^\frac32}\dx{x}{}\text{?} \end{equation*}
the substitution \(x=\sec(\theta)\) leads to: \(\int{\frac{\sec(\theta)}{\tan^2(\theta)}}\dx{\theta}\text{,}\) which we can rewrite as
\begin{align*} \int{\frac{\sec(\theta)}{\tan^2(\theta)}}\dx{\theta}\amp{}=\int\frac{1/\cos(\theta)}{\sin^2(\theta)/\cos^2(\theta)}\dx{\theta}\\ \amp{}=\int\frac{\cos(\theta)}{\sin^2(\theta)}\dx{\theta}\\ \amp{}=\int\cot(\theta)\csc(\theta)\dx{\theta}\\ \amp{}=-\csc(\theta)+C \end{align*}
(a)
Complete this integral by showing that
\begin{equation*} -\csc(\theta ) = \frac{-x}{\sqrt{x^2-1}}\text{.} \end{equation*}
(b)
Find another substitution that works, compute the integral again using your new substitution, and show that the antiderivative you obtain is equivalent to the antiderivative in part (a).
