Problem 19.3.0.3.

Did you notice that in Example 19.3.0.2 we suppressed the arbitrary constant when we computed \(v\text{?}\) In this problem we will see that this will not alter the result.

(a)

Evaluate the integral \(\int x\cos(x)\dx{x}\) letting \(u=x\) again, letting \(\dx{v}=\cos(x)\dx{x}\) as before. But this time take \(y=\sin(x)+a\) where \(a\) is an arbitrary constant. Show that we will obtain the same answer as before.

(b)

Show that in general, if we let \(\int\dx{v}=v+a\) in the integration by parts formula, then we still obtain the formula
\begin{equation*} \int u\dx{v}=uv-\int v\dx{u}. \end{equation*}
