Problem 2.2.2.6. The PFD With Linear Distinct Factors.

Let’s finish up the decomposition of
\begin{equation*} \displaystyle \frac{x}{(x-1)^2(x+2)^2}= \frac{A}{x-1} + \frac{B}{(x-1)^2} +\frac{C}{x+2} +\frac{D}{(x+2)^2} \end{equation*}
from part (b) of Problem 2.2.2.5. You should have found that \(B=\frac19\) and \(D=\frac{-2}{9}\) so that
\begin{equation*} \displaystyle \frac{x}{(x-1)^2(x+2)^2}= \frac{A}{x-1} + \frac{\frac19}{(x-1)^2} +\frac{C}{x+2} -\frac{\frac29}{(x+2)^2} \end{equation*}
When we reached this point in Example 2.2.2.3 we simply chose a value for \(x\text{.}\) We could do that here as well but we’d have to choose two values for \(x\) thereby generating a pair of equations to solve.
But we’ve already seen that there are more efficient methods available that solving a system of equations. Instead we’ll isolate all of the terms with our unknowns on one side of the equation, thus:
\begin{equation*} \displaystyle \frac{x}{(x-1)^2(x+2)^2} - \frac{A}{x-1} = \frac{\frac19}{(x-1)^2} +\frac{C}{x+2} -\frac{\frac29}{(x+2)^2} \end{equation*}
