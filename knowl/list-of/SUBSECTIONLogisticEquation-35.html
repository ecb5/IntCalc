DIGRESSION: How Did We Get a PFD.

Throughout this section we’ve been citing Theorem XXX to justify the existence of a PFD for a rational function and been using a number of shortcuts to determine the coefficients involved. It turns out that Theorem XXX can be derived from long division of polynomials, but instead of doing that, we will take the opportunity to build a PFD from the ground up though some illustrative examples and exercises. This could actually be used to compute the PFD of a given rational function in a systematic way but is not nearly as efficient as the methods employed. We are really only using it here to justify that a PFD does in fact exist for a given rational function. Some of what we have here will look familiar, but it will be included to give an entire picture of how a PFD is built starting with basics.
To begin, notice that
\begin{equation*} \frac{1}{x-a}-\frac{1}{x-b}=\frac{\left(x-b\right)-\left(x-a\right)}{\left(x-a\right)\left(x-b\right)}=\frac{a-b}{\left(x-a\right)\left(x-b\right)} \end{equation*}
Thjs says that if \(a\neq b\text{,}\) then
\begin{equation*} \frac{1}{\left(x-a\right)\left(x-b\right)}=\frac{1}{a-b}\left(\frac{1}{x-a}-\frac{1}{x-b}\right)=\frac{\frac{1}{a-b}}{x-a}+\frac{\frac{1}{b-a}}{x-b}\ \ \ \ \ \ \ \ * \end{equation*}
which agrees with what we obtained in problem 19.2.2.2.
Earlier, we also made the claim that we can find a PFD for three or more distinct linear factors, and that this follows from Theorem XXX. We can utilize the formula * to obtain this as well and we can utilize what we already did. Suppose \(a,\ b,\ c\) are distinct real numbers. Then
\begin{equation*} \frac{1}{\left(x-a\right)\left(x-b\right)\left(x-c\right)}=\frac{1}{a-b}\left(\frac{1}{x-a}-\frac{1}{x-b}\right)\left(\frac{1}{x-c}\right)=\frac{1}{a-b}\left(\frac{1}{\left(x-a\right)\left(x-c\right)}-\frac{1}{\left(x-b\right)\left(x-c\right)}\right) \end{equation*}
\textbf{Problem.} a. Use our formula * on the two fractions in the parentheses above to obtain the PFD
\begin{equation*} \frac{1}{\left(x-a\right)\left(x-b\right)\left(x-c\right)}=\frac{\frac{1}{\left(a-b\right)\left(a-c\right)}}{x-a}+\frac{\frac{1}{\left(b-a\right)\left(b-c\right)}}{x-b}+\frac{\frac{1}{\left(c-b\right)\left(c-a\right)}}{x-c} \end{equation*}
which is what we obtained in Problem 19.2.2.8.
b. Suppose we have four distinct real numbers \(a,b,c,d\text{.}\) Using the result obtained in part a, we obtain
\begin{equation*} \frac{1}{\left(x-a\right)\left(x-b\right)\left(x-c\right)\left(x-d\right)}=\left(\frac{\frac{1}{\left(a-b\right)\left(a-c\right)}}{x-a}+\frac{\frac{1}{\left(b-a\right)\left(b-c\right)}}{x-b}+\frac{\frac{1}{\left(c-b\right)\left(c-a\right)}}{x-c}\right)\left(\frac{1}{x-d}\right) \end{equation*}
Use * to show that we have the PFD
\begin{equation*} \frac{1}{\left(x-a\right)\left(x-b\right)\left(x-c\right)\left(x-d\right)}=\frac{\frac{1}{\left(a-b\right)\left(a-c\right)(a-d)}}{x-a}+\frac{\frac{1}{\left(b-a\right)\left(b-c\right)\left(b-d\right)}}{x-b}+\frac{\frac{1}{\left(c-a\right)\left(c-b\right)\left(c-d\right)}}{x-c}+\frac{\frac{1}{\left(a-b\right)\left(a-c\right)\left(d-a\right)}+\frac{1}{\left(b-a\right)\left(b-c\right)\left(d-b\right)}+\frac{1}{\left(c-a\right)\left(c-b\right)\left(d-c\right)}}{x-d} \end{equation*}
Compare this result to what you get using the HCUM. [You can see why we are using * to show the existence of a PFD, instead of using it to determine the coefficients.]
So far, we’ve looked at the existence of a PFD for a rational function whose numerator is one. As we indicated in example 19.2.3.1, the case where the numerator is not one can be handled by factoring it out and applying the appropriate substitutions. We will utilize the following as an illustrative example, but it should be clear that this technique could be generalized.
\textbf{Example.} Find the PFD of \(\frac{2x^2+x+1}{\left(x-4\right)\left(x-2\right)}\text{.}\)
Notice that we can write this as
\begin{equation*} \frac{2x^2+x+1}{\left(x-4\right)\left(x-2\right)}=\left(2x^2+x+1\right)\left(\frac{1}{\left(x-4\right)\left(x-2\right)}\right) \end{equation*}
\begin{equation*} =\left(2x^2+x+1\right)\left(\frac{\frac{1}{2}}{x-4}-\frac{\frac{1}{2}}{x-2}\right) \end{equation*}
\begin{equation*} =\frac{1}{2}\left(\frac{2x^2+x+1}{x-4}-\frac{2x^2+x+1}{x-2}\right) \end{equation*}
If we let \(y=x-4\) and \(z=x-2\text{,}\) we get
\begin{equation*} \frac{2x^2+x+1}{\left(x-4\right)\left(x-2\right)}=\frac{1}{2}\left(\frac{2{\left(y+4\right)}^2+\left(y+4\right)+1}{y}-\frac{2{\left(z+2\right)}^2+\left(z+2\right)+1}{z}\right) \end{equation*}
\begin{equation*} =\frac{1}{2}\left(\frac{2y^2+17y+37}{y}-\frac{2z^2+9z+11}{z}\right) \end{equation*}
\begin{equation*} =\frac{1}{2}\left(2y+17+\frac{37}{y}-2z-9-\frac{11}{z}\right) \end{equation*}
\begin{equation*} =\frac{1}{2}\left(2y-2z+8+\frac{37}{y}-\frac{11}{z}\right) \end{equation*}
\begin{equation*} =y-z+4+\frac{\frac{37}{2}}{y}-\frac{\frac{11}{2}}{z} \end{equation*}
\begin{equation*} =x-4-x+2+4+\frac{\frac{37}{2}}{x-4}-\frac{\frac{11}{2}}{x-2}=2+\frac{\frac{37}{2}}{x-4}-\frac{\frac{11}{2}}{x-2} \end{equation*}
\textbf{}
\textbf{Problem.} Apply the same technique to determine the PFD of
\begin{equation*} \frac{x^3}{\left(x-4\right)\left(x-2\right)} \end{equation*}
As we said, this technique can be generalized to any rational function whose denominator is a product of distinct linear factors. It can also be used if there is a repeated linear factor. Again, it suffices to do this where the numerator is one.
\textbf{Example.} Let \(a,b\) be distinct real numbers. Then
\begin{equation*} \frac{1}{{\left(x-a\right)}^2\left(x-b\right)}=\frac{1}{x-a}\left(\frac{1}{\left(x-a\right)\left(x-b\right)}\right) \end{equation*}
\begin{equation*} =\frac{1}{x-a}\left(\frac{1}{a-b}\left(\frac{1}{x-a}-\frac{1}{x-b}\right)\right) \end{equation*}
\begin{equation*} =\frac{1}{a-b}\left(\frac{1}{{\left(x-a\right)}^2}-\frac{1}{\left(x-a\right)\left(x-b\right)}\right) \end{equation*}
\begin{equation*} =\frac{1}{a-b}\left(\frac{1}{{\left(x-a\right)}^2}-\frac{1}{a-b}\left(\frac{1}{x-a}-\frac{1}{x-b}\right)\right) \end{equation*}
\begin{equation*} =\frac{\frac{1}{a-b}}{{\left(x-a\right)}^2}-\frac{\frac{1}{{\left(a-b\right)}^2}}{x-a}+\frac{\frac{1}{{\left(a-b\right)}^2}}{x-b} \end{equation*}
Again, don’t try to memorize such a formula. The purpose of this is to show that such a PFD exists. There are more efficient ways to compute the coefficients as you have seen.
\textbf{Problem.} Use the fact that
\begin{equation*} \frac{1}{{\left(x-a\right)}^3\left(x-b\right)}=\frac{1}{x-a}\left(\frac{1}{{\left(x-a\right)}^2\left(x-b\right)}\right)=\frac{1}{x-a}\left(\frac{\frac{1}{a-b}}{{\left(x-a\right)}^2}-\frac{\frac{1}{{\left(a-b\right)}^2}}{x-a}+\frac{\frac{1}{{\left(a-b\right)}^2}}{x-b}\right) \end{equation*}
to show that
\begin{equation*} \frac{1}{{\left(x-a\right)}^3\left(x-b\right)}=\frac{\frac{1}{a-b}}{{\left(x-a\right)}^3}-\frac{\frac{1}{{\left(a-b\right)}^2}}{{\left(x-a\right)}^2}+\frac{\frac{1}{{\left(a-b\right)}^3}}{x-a}-\frac{\frac{1}{{\left(a-b\right)}^3}}{x-b} \end{equation*}
\textbf{Problem. }Use the fact that
\begin{equation*} \frac{1}{{\left(x-a\right)}^2{\left(x-b\right)}^2}=\left(\frac{1}{{\left(x-a\right)}^2\left(x-b\right)}\right)\left(\frac{1}{x-b}\right)=\left(\frac{\frac{1}{a-b}}{{\left(x-a\right)}^2}-\frac{\frac{1}{{\left(a-b\right)}^2}}{x-a}+\frac{\frac{1}{{\left(a-b\right)}^2}}{x-b}\right)\left(\frac{1}{x-b}\right) \end{equation*}
to show that
\begin{equation*} \frac{1}{{\left(x-a\right)}^2{\left(x-b\right)}^2}=\frac{\frac{1}{{\left(a-b\right)}^2}}{{\left(x-a\right)}^2}-\frac{\frac{2}{{\left(a-b\right)}^3}}{x-a}+\frac{\frac{2}{{\left(a-b\right)}^3}}{x-b}+\frac{\frac{1}{{\left(a-b\right)}^2}}{{\left(x-b\right)}^2} \end{equation*}
We’ve taken care of the existence of a PFD where the denominator is a product of (possibly repeated) linear factors. What about an irreducible quadratic factor? As we saw before, if we utilize complex numbers, then these can be factored into linear factors as well. The beauty is that in the end, we will have real numbers as our coefficients. To see this, we need to look a little into complex number arithmetic. Perhaps even more surprising is that all this hinges on the notion of a complex conjugate. Given a complex number \(z=a+bi\text{,}\) its complex conjugate is given by \(\overline{\ z\ }=a-bi\text{.}\) The following are facts about complex conjugates.
\textbf{Theorem X.} Let \(z=a+bi\) and \(w=c+di\text{,}\) then
\begin{enumerate} \item \(\overline{z+w}=\overline{\ z\ }+~\overline{w}\)
\item \(\overline{z\cdot w}=\overline{\ z\ }\cdot \overline{w}\)
\item \(z+\overline{\ z\ }=2a\) which is a real number
\item \(z-\overline{\ z\ }=2bi\)
\item \(z\cdot \overline{\ z\ }=a^2+b^2\) which is a nonnegative real number. \end{enumerate}
\textbf{Problem.} Prove Theorem X
As was said previously, the Fundamental Theorem of Algebra says that every nonconstant polynomial with real coefficients has at least one root which could be a complex number. A consequence of this is that any nonconstant polynomial and be factored entirely into linear factors where some of the factors may involve complex roots. Specifically, if \(p\left(x\right)=a_0+a_1x+a_2x^2+\dots +a_nx^n\) is a real polynomial with roots \(r_1,\ r_2,\ \dots ,\ r_n\text{,}\) (some roots possibly repeated, some possibly complex), then
\begin{equation*} p\left(x\right)=a_n\left(x-r_1\right)\left(x-r_2\right)\dots (x-r_n) \end{equation*}
The interesting thing about \(p\left(x\right)\) is that if it has a root which is a complex number \(z\) then \(\overline{\ z\ }\ \)is also a root.
\textbf{Problem.} Prove that if \(z\) is a root of \(p\left(x\right)=a_0+a_1x+a_2x^2+\dots +a_nx^n\text{,}\) then \(\overline{\ z\ }\) is a root.
[Hint: If z is a root, then \(a_0+a_1z+a_2z^2+\dots +a_nz^n=0\text{.}\) Take the conjugate of this and use properties in Theorem X. Also note that any real number is its own conjugate.]
The first consequence of this is that any real polynomial which is irreducible must be of degree 1 or 2. To see this, suppose we have a real polynomial \(p(x)\) of degree greater than 2. If one of the roots of this polynomial is real, say \(r\text{,}\) then we can write \(p\left(x\right)=\left(x-r\right)q(x)\) for some polynomial \(q(x)\) whose degree is at least two. This says \(p\left(x\right)\ \)is not irreducible. If \(p(x)\) has a complex root \(z\text{,}\) then \(\overline{\ z\ }\ \)is also a root, and we can write \(p\left(x\right)=\left(x-z\right)\left(x-\overline{\ z\ }\right)q(x)\) where \(q\left(x\right)\ \)has degree at least one.
\textbf{Problem.} Multiply \(\left(x-z\right)\left(x-\overline{\ z\ }\right)\) and conclude that it is a polynomial with real coefficients. This says that \(p\left(x\right)\) is not irreducible.
With these preliminaries established, suppose we have the rational function
\begin{equation*} \frac{1}{\left(x^2+ax+b\right)\left(x-c\right)} \end{equation*}
where \(x^2+ax+b\) is an irreducible real polynomial and \(c\) is a real number. If we factor \(x^2+ax+b\) over the complex numbers, we get
\begin{equation*} \frac{1}{\left(x^2+ax+b\right)\left(x-c\right)}=\frac{1}{\left(x-z\right)\left(x-\overline{\ z\ }\right)\left(x-c\right)} \end{equation*}
Note that everything we’ve done so far works over the complex numbers, so utilizing
\begin{equation*} \frac{1}{\left(x-a\right)\left(x-b\right)\left(x-c\right)}=\frac{\frac{1}{\left(a-b\right)\left(a-c\right)}}{x-a}+\frac{\frac{1}{\left(b-a\right)\left(b-c\right)}}{x-b}+\frac{\frac{1}{\left(c-b\right)\left(c-a\right)}}{x-c} \end{equation*}
we have
\begin{equation*} \frac{1}{\left(x-z\right)\left(x-\overline{\ z\ }\right)\left(x-c\right)}=\frac{\frac{1}{\left(z-\overline{\ z\ }\right)\left(z-c\right)}}{x-z}+\frac{\frac{1}{\left(\overline{\ z\ }-z\right)\left(\overline{\ z\ }-c\right)}}{x-\overline{\ z\ }}+\frac{\frac{1}{\left(c-\overline{\ z\ }\right)\left(c-z\right)}}{x-c} \end{equation*}
\textbf{Problem.} Show that \(\frac{1}{\left(c-\overline{\ z\ }\right)\left(c-z\right)}\) is a real number.
[Hint: Note that \(c-\overline{\ z\ }=\overline{c-z}\text{.}\)]
\textbf{Problem.} a. Show that
\begin{equation*} \frac{\frac{1}{\left(z-\overline{\ z\ }\right)\left(z-c\right)}}{x-z}+\frac{\frac{1}{\left(\overline{\ z\ }-z\right)\left(\overline{\ z\ }-c\right)}}{x-\overline{\ z\ }}=\frac{-\frac{1}{\left(z-c\right)\left(\overline{\ z\ }-c\right)}x+\left(\frac{z}{\left(z-\overline{\ z\ }\right)\left(\overline{\ z\ }-c\right)}+\frac{\overline{\ z\ }}{\left(\overline{\ z\ }-z\right)\left(z-c\right)}\right)}{\left(x^2+ax+b\right)} \end{equation*}
b. We already know that \(-\frac{1}{\left(z-c\right)\left(\overline{\ z\ }-c\right)}\) is a real number. Show that
\begin{equation*} \frac{z}{\left(z-\overline{\ z\ }\right)\left(\overline{\ z\ }-c\right)}+\frac{\overline{\ z\ }}{\left(\overline{\ z\ }-z\right)\left(z-c\right)} \end{equation*}
is a real number also.
[Hint: To avoid a lot of messy algebra, notice that these are conjugates.]
Putting this all together, this says that
\begin{equation*} \frac{1}{\left(x^2+ax+b\right)\left(x-c\right)}=\frac{Ax+B}{x^2+ax+b}+\frac{C}{x-c} \end{equation*}
where \(A,B,C\) are real numbers. This is the PFD we were seeking. This same idea will work for any denominator which is factored into linear or quadratic irreducible factors.
