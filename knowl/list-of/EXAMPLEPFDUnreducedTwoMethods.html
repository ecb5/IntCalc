Example 19.2.2.1.
Heaviside’s method, as we have seen it so far, will not work on the rational function \(\frac{x^2}{(x-2)(x-3)}\text{.}\) If you try to apply it you’ll get
\begin{equation*} \frac{x^2}{(x-2)(x-3)} = \frac{-4}{x-2}+\frac{9}{x-3} \end{equation*}
which is wrong. Try it and see.
The difficulty here is that, as a practical matter, the PFD requires that we want the degree of the numerator of our rational function to be strictly less than the degree of the denominator. (This is somewhat analogous to putting a fraction in “lowest terms.”) In this example the degree of the numerator is \(\deg\left((x-2)(x-3)\right)=\deg\left(x^2-5x+6\right)=2\) which is the same as the degree of the denominator: \(\deg\left(x^2\right)=2\text{.}\) So the PFD, as stated so far, doesn’t apply.
At first this looks like it will be a major obstacle but it really isn’t too bad. There are two ways we can address the issue.
Dividing First: The first is simply to do the polynomial division indicated. In practice that means multiplying out the denominator first: \((x-2)(x-3)=x^2-5x+6\) and then dividing:
\begin{align} \frac{x^2}{x^2-5x+6} \amp{}= 1+ \frac{5x-6}{(x-2)(x-3)}\notag\\ \amp{}=1+\frac{5x-6}{(x-2)(x-3)}\tag{19.43} \end{align}
From there we can compute the PFD of \(\frac{5x-6}{(x-2)(x-3)}\) as before, finally obtaining
\begin{equation*} \frac{x^2}{x^2-5x+6} = 1+\frac{-4}{x-2}+\frac{9}{x-3}\text{.} \end{equation*}
Dividing Last: The second is to factor out the numerator:
\begin{equation*} \frac{x^2}{(x-2)(x-3)} = x^2 \left[ \frac{1}{(x-2)(x-3)} \right] \text{,} \end{equation*}
compute the PFD of \(\frac{1}{(x-2)(x-3)}\) as before. This gives
\begin{align} \frac{x^2}{(x-2)(x-3)} \amp{}= x^2 \left[ \frac{-1}{x-2}+\frac{1}{x-3} \right]\notag\\ \amp{}=\frac{-x^2}{x-2}+\frac{x^2}{x-3}\tag{19.44}\\ \amp{}= -x-2-\frac{4}{x-2} +x+3+\frac{9}{x-3}\notag\\ \amp{}=1-\frac{4}{x-2}+\frac{9}{x-3}.\notag \end{align}
These two methods are equivalent so which to use is entirely up to you. We (the authors) prefer dividing last because it doesn’t require that we factor and un–factor the denominator, and because we only ever have to divide by a linear expression, which we find less burdensome.
However, we do advise that you pick one method that you like and use it consistently, at least until you have fully mastered the Partial Fraction Decomposition.
