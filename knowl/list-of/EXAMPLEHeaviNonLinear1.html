Example 2.2.3.4.
Heaviside’s Cover–up Method will also fail on this function:
\begin{equation*} \frac{x^2+x-3}{x^2(x-1)}\text{.} \end{equation*}
If you apply it you’ll get
\begin{equation*} \frac{x^2+x-3}{x^2(x-1)} = \frac{3}{x^2} + \frac{-1}{x-1} \end{equation*}
which is incorrect. Check it and see.
The problem here is the factor \(x^2\) in the denominator. If you look back at the problems and examples we’ve looked at so far you’ll see that each distinct factor which appeared in the denominator of the fraction on the left side also appeared in one of the denominators on the right side. In this case the distinct factors are \(x^2\) and \(x-1\) so the decomposition will be
\begin{equation*} \frac{x^2+x-3}{x^2(x-1)} = \frac{\text{something}}{x^2} + \frac{C}{x-1} \text{.} \end{equation*}
But now the question is, “What is the something?” Up until now none of the distinct factors in the denominator appeared with a power higher than one. So a factor of \(x^2\) is altogether new and different.
Looking back at our earlier, and simpler, examples we see that every case the terms on the left were rational functions where the degree of the numerator (which was always \(0\) in those examples) was one less than the degree of the denominator (which was always \(1\) in those examples). If we keep that pattern here then it appears that the decomposition must be
\begin{equation*} \frac{x^2+x-3}{x^2(x-1)} = \frac{Ax+B}{x^2} + \frac{C}{x-1} \end{equation*}
because
\begin{equation*} \deg (Ax+B)=1 \end{equation*}
and
\begin{equation*} \deg(x^2) =2\text{.} \end{equation*}
We can break this up a bit more by observing that
\begin{equation*} \frac{Ax+B}{x^2} =\frac{A}{x}+\frac{B}{x^2} \end{equation*}
so we need to find \(A\text{,}\) \(B\text{,}\) and \(C\) where
\begin{equation*} \frac{x^2+x-3}{x^2(x-1)} = \frac{A}{x}+\frac{B}{x^2} + \frac{C}{x-1} \text{.} \end{equation*}
The values \(B=3\text{,}\) and \(C=-1\) can still be found using the HCUM, so we have
\begin{equation} \frac{x^2+x-3}{x^2(x-1)} = \frac{A}{x} + \frac{3}{x^2}- \frac{1}{x-1}\text{.}\tag{2.43} \end{equation}
To find \(A\) recall that equation (2.43) is an identity so it is true regardless of the value of \(x\text{.}\) Since \(A\) is the only parameter left we can find it by setting \(x=2\) (or anything else). This gives \(\frac{2}{2^2(2-1)} = \frac{A}{2} + \frac{3}{2^2}- \frac{1}{2-1} \text{,}\) or \(A=2\text{.}\) So the full decomposition is
\begin{equation*} \frac{x^2+x-3}{x^2(x-1)} = \frac{2}{x}+ \frac{3}{x^2}-\frac{1}{x-1}\text{.} \end{equation*}
