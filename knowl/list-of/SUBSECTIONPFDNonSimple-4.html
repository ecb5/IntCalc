Example 19.2.2.3.

Heaviside’s method will also fail on this function:
\begin{equation*} \frac{x^2+x-3}{x^2(x-1)}\text{.} \end{equation*}
If you apply it you’ll get
\begin{equation*} \frac{x^2+x-3}{x^2(x-1)} = \frac{3}{x^2} + \frac{-1}{x-1} \end{equation*}
which is also incorrect. Check it and see.
The problem here is the factor \(x^2\) in the denominator. If you look back at the problems we’ve done so far you’ll see that in every case the fractions in our decomposition (the right hand side) included a term with every possible factor of the denominator of our rational function (the left hand side). In this case clearly both \(x^2\) and \(x-1\) are factors of \(x^2(x-1)\text{,}\) so the decomposition
\begin{equation*} \frac{x}{x^2(x-1)} = \frac{A}{x^2} + \frac{B}{x-1} \end{equation*}
is good as far as it goes. But \(x\) is also a factor so we need to find the constants \(A\text{,}\) \(B\text{,}\) and \(C\) in the decomposition
\begin{equation*} \frac{x}{x^2(x-1)} = \frac{A}{x} + \frac{B}{x^2}+ \frac{C}{x-1}\text{.} \end{equation*}
The correct values are \(A=2\text{,}\) \(B=3\text{,}\) and \(C=-1\text{.}\) Notice that Heaviside’s method correctly identified both \(B\) and \(C\) so all we have to do is find \(A\text{.}\) There is hope.
To find \(A\) we set up
\begin{equation*} \frac{x}{x^2(x-1)} = \frac{A}{x} + \frac{B}{x^2}+ \frac{C}{x-1} \end{equation*}
as before and compute \(B=3\) and \(C=-1\) as before. So
\begin{equation*} \frac{x}{x^2(x-1)} = \frac{A}{x} + \frac{3}{x^2}- \frac{1}{x-1}\text{.} \end{equation*}
Notice that this last equation is still and indentity and so it is true regardless of the value of \(x\text{.}\) The obvious thing to do then is simply set \(x\) equal to any number except zero or one (why?) and solve for \(A\text{.}\)
Taking \(x=2\) we have
\begin{gather*} \frac{2}{2^2(2-1)} = \frac{A}{2} + \frac{3}{2^2}- \frac{1}{2-1} \end{gather*}
clear all of the fractions and isolate \(A\) on one side of the equation. Since \(A\) is a constant everything involving an \(x\) will cancel and the value of \(A\) will be left.
Solving for \(A\) we have
\begin{equation*} \frac{x}{x(x-1)} - \frac{3}{x}+ \frac{x}{x-1} = A\text{.} \end{equation*}
