Problem 2.2.1.1.

(a)
Use equation (2.31) to find \(y\) for the logistic growth model with the initial condition \(y(0)=0\)
\begin{equation*} \dfdx{y}{t}=0.01y\left(1-\frac{y}{100}\right)\hskip5mm y(0)=0 \end{equation*}
for the case when \(y\lt 100\text{.}\)
(b)
Solve the same problem for the case where \(y\gt 100\text{.}\)
Comment.
This is a spot where you need to look at the absolute value in the logarithm. We told you it comes up occasionally.
