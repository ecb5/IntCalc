Example 1.1.0.6.

Separating variables is not new. This is how you were taught to solve algebraic equations. For example, suppose we want to solve the algebraic equation
\begin{equation} y^3-x^2y - 6y=0\tag{1.6} \end{equation}
for \(y\text{.}\) One obvious solution is \(y=0\text{.}\) This is a valid solution, but it is uninteresting, so we will not consider it. Assuming that \(y\neq 0\text{,}\) we can divide by \(y\) obtaining
\begin{equation*} y^2-x^2-6=0 \end{equation*}
At this point we separate the variables by rearranging the terms algebraically so that all of the terms involving \(y\) are on on one side, and all of the terms involving \(x\) (or involving neither \(x\) nor \(y\)) on the other
\begin{equation*} y^2=x^2+6\text{.} \end{equation*}
It is clear that the expression on the right is positive so we can take the square root of each side and obtain \(y=\pm \sqrt{x^2+6}\text{.}\)
Notice that it was not immediately clear that separating the variables would be helpful but once it was done, the next step was clear.
It is similarly not clear that separating the variables in equation (1.5) will help, but let’s try it and see what happens. Separating the variables, we see that
\begin{equation*} \frac{\dx{y}}{y}=x\dx{x} \end{equation*}
which says that the differentials on either side of the equation are the same. Integrating (summing) both sides gives
\begin{equation} \int{\frac{1}{y}} \dx{y}=\int{x\dx{x}}\text{.}\tag{1.7} \end{equation}
