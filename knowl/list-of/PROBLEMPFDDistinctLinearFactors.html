Problem 2.2.2.5.
Use the HCUM to compute the coefficients of the PFD indicated.
(a)
Find \(B\text{,}\) \(C\text{,}\) and \(F\text{.}\)
\begin{align*} \frac{1}{x^2(x-1)(x-3)^3}=\amp{}\frac{A}{x} + \frac{B}{x^2} + \frac{C}{x-1}\\ \amp{}+ \frac{D}{x-3} + \frac{E}{(x-3)^2} + \frac{F}{(x-3)^3} \end{align*}
(b)
Find \(B\) and \(D\text{.}\)
\begin{equation*} \displaystyle \frac{x}{(x-1)^2(x+2)^2}= \frac{A}{x-1} + \frac{B}{(x-1)^2} +\frac{C}{x+2} +\frac{D}{(x+2)^2} \end{equation*}
(c)
Find \(C\text{.}\)
\begin{equation*} \frac{x-2}{x^3}=\frac{A}{x}+\frac{B}{x^2}+\frac{C}{x^3} \end{equation*}
(d)
In each of these problems we only asked you to find the coefficients that are attainable using the HCUM. The others will have to be computed by the method we’ll describe in the next example.
Explain as clearly as you can from the evidence above which kinds of coefficients we can compute using the HCUM.
