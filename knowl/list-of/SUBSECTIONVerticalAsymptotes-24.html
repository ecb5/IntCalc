One-Sided Infinite Limits vs. Infinite Limits.

Finally, does the knowledge that
\begin{align*} \rlimit{x}{0} { \frac{1}{x} }=\infty \amp{}\amp{} \text{and} \amp{}\amp{} \llimit{x}{0}{\frac{1}{x} }=-\infty \end{align*}
tell us anything about \(\limit{x}{0}{\frac{1}{x}}\text{?}\) (Notice that this is not a one-sided limit.)
Well, yes. Sort of. It tells us that the expression \(\limit{x}{0}{\frac{1}{x}}\) is without meaning; that it is undefined.
Here’s why.
  1. It cannot be true that \(\limit{x}{0}{\frac{1}{x}}=\infty\) because we can always choose a small positive number \(a\) which guarantees that \(f(a)\) is greater than any number, \(B\text{,}\) we might choose.
  2. It cannot be true that \(\limit{x}{0}{\frac{1}{x}}=-\infty\) because we can always choose a small negative number \(a\) which guarantees that \(f(a)\) is less than than any number, \(B\text{,}\) we might choose.
  3. It cannot be true that \(\limit{x}{0}{\frac{1}{x}}\) is equal to some real number because we can always choose a small positive number \(a\) which guarantees that \(f(a)\) is greater than any real number we might choose.
That is, our limit cannot be positively infinite (part (a), it cannot be negatively infinite (part (b), and it cannot be a real number (part (c)).
What we have here is Sherlock Holmes’ Maxim 9.2.0.17 taken to its extreme. Instead of eliminating all but one possibility we have eliminated all possibilities. Therefore we forced to conclude that \(\limit{x}{0}{\frac{1}{x}}\) does not exist.
On the other hand if
\begin{equation*} \rlimit{x}{0} { f(x) }=\infty =\llimit{x}{0}{f(x) } \end{equation*}
we can unambiguously define
\begin{equation*} \limit{x}{0} { f(x) } \end{equation*}
to be \(\infty{}\text{.}\)
And if
\begin{equation*} \rlimit{x}{0} { f(x) }=-\infty =\llimit{x}{0}{f(x) } \end{equation*}
we can unambiguously define
\begin{equation*} \limit{x}{0} { f(x) } \end{equation*}
to be \(-\infty\text{.}\)
Example 17.2.4.11.
For example \(\limit{x}{0}{\frac{1}{x^2}} = \infty \) since
\begin{equation*} \rlimit{x}{0}{\frac{1}{x^2}} = \infty =\llimit{x}{0}{\frac{1}{x^2}}\text{.} \end{equation*}
Problem 17.2.4.12.
(a)
Use Definition 17.2.4.2 to show that
\begin{equation*} \rlimit{x}{0}{\frac{1}{x^2}} = \infty \end{equation*}
(b)
Use the appropriate definition from Problem 17.2.4.10 to show that
\begin{equation*} \llimit{x}{0}{\frac{1}{x^2}} = \infty \end{equation*}
Thus we have the following definition.
Definition 17.2.4.13. Positive, Infinite Limits.
Suppose \(f(x)\) is defined near \(x=a\text{.}\) Then we say
\begin{equation*} \limit{x}{a}{f(x)=\infty } \end{equation*}
if and only if
\begin{equation*} \rlimit{x}{a}{f(x)=\infty }=\llimit{x}{a}{ f(x) }\text{.} \end{equation*}
Problem 17.2.4.14.
Prove each of the following statements.
(a)
\(\limit{x}{0}{\frac{1}{x^4}}=\infty \)
(b)
\(\limit{x}{0}{\frac{-1}{x^4}}=-\infty \)
(c)
\(\limit{x}{0}{\frac{1}{x^3}} \) does not exist.
(d)
\(\limit{x}{3}{\frac{3}{(x-3)}}\) does not exist.

Problem 17.2.4.16.
Provide a definition of \(\limit{x}{a}{ f(x)}=-\infty \) similar to Definition 17.2.4.13 and use your definition to show that
\begin{equation*} \limit{x}{a}{f(x)}=-\infty \end{equation*}
if and only if
\begin{equation*} \limit{x}{a}{(-f(x))}=\infty\text{.} \end{equation*}
Problem 17.2.4.17. Infinite Limits: An Alternate Definition.
Suppose \(f(x)\) is defined near \(x=a\text{.}\) Show that if for every real number \(B>0\text{,}\) there is a \(\delta >0\) such that
\begin{equation*} f\left(x\right)\gt B \end{equation*}
whenever
\begin{equation*} a-\delta \lt x\lt a+\delta \text{ and } x\neq a\text{,} \end{equation*}
then
\begin{equation*} \limit{x}{a}{f(x)}=\infty \text{.} \end{equation*}
It is common to take the statement in Problem 17.2.4.17 as the definition of \(\limit{x}{a}{f(x)}=\infty \) and then to prove Definition 17.2.4.13 as a theorem. You did this in Problem 17.2.4.17. Logically the two statements are equivalent so it doesn’t matter which is taken as the definition. Each statement can be used to prove the other as a theorem.
Problem 17.2.4.18.
Complete this equivalence by showing that if
\begin{equation*} \rlimit{x}{a}{f(x)} = \infty = \llimit{x}{a}{f(x)} \end{equation*}
then for every real number \(B>0\text{,}\) there is a \(\delta \gt 0\) such that \(f\left(x\right)\gt B\) whenever \(a-\delta \lt x\lt a+\delta \) and \(x\neq a\text{.}\)
Finally, notice that in Problem 17.2.4.17 , we had the conditions \(a-\delta \lt x\lt a+\delta \) and \(x\neq a\text{.}\) This can be more succinctly written as \(0\lt \left|x-a\right|\lt \delta \text{.}\) Keep this in mind as you read the next section on limits that don’t involve infinity.
