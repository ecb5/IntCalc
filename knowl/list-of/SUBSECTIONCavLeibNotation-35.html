Problem 21.1.2.6. Find the Pattern.

(a)
Show that each of the following statements is true.
  1. \(\displaystyle \int_{x=0}^1x \dx{x}=\frac12 \)
  2. \(\displaystyle \int_{x=0}^1x^2 \dx{x}=\frac13 \)
  3. \(\displaystyle \int_{x=0}^1x^3 \dx{x}=\frac14 \)
  4. \(\displaystyle \int_{x=0}^1x^4 \dx{x}=\frac15 \)
(b)
Use the pattern you observed in part (a) to predict the value of
\begin{align*} \int_{x=0}^1x^\alpha \dx{x}\amp{}\amp{} \text{ where } \alpha \gt 0 \end{align*}
then show that your prediction is true.
(c)
What happens if \(\alpha \le 0\text{?}\)
